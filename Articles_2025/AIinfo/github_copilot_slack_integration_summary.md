# ［速報］GitHub Copilot Coding AgentがSlackとの連携機能を搭載

**🔗 URL**: [［速報］GitHub Copilot Coding AgentがSlackとの連携機能を搭載](https://www.publickey1.jp/blog/25/github_copilot_coding_agentslack.html)  
**📅 Date**: 2025-10-29

## 📄 Original Content

GitHubは年次イベント「GitHub Universe 2025」において、GitHub Copilot Coding AgentとSlackの連携機能を発表した。これまでJira、Microsoft Teams、Azure Boards、Raycastなどと連携していたが、新たにSlackおよびLinearとの統合が可能になった。デモ動画では、Slackのスレッド上でGitHubにバグフィックスを指示すると、GitHub Copilot Coding Agentが自動的に対応し、その結果がGitHub側にも反映される様子が示されている。

## 🎯 Summary

- **Slack連携の新機能搭載**: GitHub Copilot Coding AgentがSlackおよびLinearとの連携機能を追加し、チャットベースの開発ワークフロー統合を実現
- **既存連携の拡張**: Jira、Microsoft Teams、Azure Boards、Raycastに加え、コミュニケーションツールからの直接的なコーディングエージェント操作が可能に
- **シームレスなバグ修正フロー**: Slackスレッド上でのバグフィックス指示が自動的にGitHub Copilot Coding Agentに伝達され、実装まで完結
- **双方向同期**: Slackでの指示とGitHub上での実装結果が相互に反映され、チーム全体での可視性を確保
- **開発効率の向上**: コミュニケーションツールから離れることなくコーディングタスクを実行できる統合開発環境の実現

## 📊 連携ツール一覧

| カテゴリ | ツール名 | 連携機能 |
|---|---|---|
| プロジェクト管理 | Jira | タスク管理・イシュー連携 |
| プロジェクト管理 | Linear | タスク管理・イシュー連携（新規） |
| プロジェクト管理 | Azure Boards | タスク管理 |
| コミュニケーション | Microsoft Teams | チャット連携 |
| **コミュニケーション** | **Slack** | **チャット連携（新規）** |
| ランチャー | Raycast | コマンドランチャー連携 |

## 💭 My Notes

- 品質保証の観点から、Slack連携によるバグトラッキングとフィックス実装の一気通貫フローは、修正時間の短縮と追跡性向上に貢献する可能性
- チーム内コミュニケーションツール上で直接コーディングエージェントを操作できることで、問題報告から修正完了までのリードタイム削減が期待される
- 製品開発において、テスト結果のSlack通知と自動修正指示の組み合わせにより、CI/CDパイプラインの高度な自動化が実現可能
- ただし、エージェントが生成したコードの品質検証プロセスは依然として人間のレビューが必要で、過度な自動化への依存はリスクを伴う
- 子供たちのプログラミング学習において、Slackのような馴染みのあるツールから学習支援を受けられる形態は、学習ハードルの低減に効果的
