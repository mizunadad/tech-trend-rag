# LangGraph + Claude API 実例とコスト情報

**作成日**: 2024-12-02

## 概要

個人でClaude APIを契約した後、LangGraphと組み合わせて実際にどれくらいのコストで何ができるのか、実例とともに整理したドキュメント。

## Claude API 料金体系（個人契約）

**Claude 3.5 Sonnet**
- 入力: $3/1M tokens
- 出力: $15/1M tokens

**Claude 3 Haiku**
- 入力: $0.25/1M tokens
- 出力: $1.25/1M tokens

## 実際のコスト感（実例ベース）

### 小規模テスト
- **簡単な自動化タスク（10-20回実行）**: $0.50以下
- **動作確認・学習用途**: $1-2程度

### 実用レベル
- **データ分析レポート作成（50回実行）**: $1-2程度
- **月間の業務自動化（500回実行）**: $10-20程度
- **3エージェント連携システム（100回実行）**: $2-3程度

### 品質保証業務での想定
- **サプライヤー情報収集→分析→レポート生成の自動化**: 月$20-30程度

## おすすめ実例リソース

### 1. LangGraph公式チュートリアル
- **URL**: https://python.langchain.com/docs/langgraph/tutorials/
- **内容**: 基本から段階的に学べる公式チュートリアル
- **コスト**: 最初の実験なら$1以下で十分

- **内容**: Claude Sonnet 3.5を使った3エージェント実装
- **構成**: リサーチャー、アナリスト、ライター

### 3. GitHub実例プロジェクト
- **URL**: LangChain公式GitHub

## 実装の基本構成

### 必要なもの
1. Python環境
2. LangGraph（無料、オープンソース）
3. Claude API key（個人契約済み）

### インストール
```bash
pip install langgraph langchain-anthropic
```

### 基本的なワークフロー例
1. **エージェント1**: 情報収集（リサーチャー）
2. **エージェント2**: データ分析（アナリスト）
3. **エージェント3**: レポート生成（ライター）
