# CASTトレンド分析 4-2 AI（人工知能）

## Summary - 5つの要点

1. **自動運転2.0（AV2.0）への劇的進化**
   - 従来のルールベース（AV1.0）から生成AI活用のエンド・ツー・エンド（E2E）手法（AV2.0）へパラダイムシフト
   - テスラFSD V.12（2023年）がE2Eモデル実装を開始、中国XPeng等が追随
   - 認知から操作まで全てをAIが自律的に判断、人間によるルール定義を大幅削減
   - 世界自動運転車市場は2021年240億ドルから2026年620億ドルへ2.6倍成長予測

2. **生成AI×仮想環境シミュレーションの革命**
   - カナダWaabiが生成AI仮想空間「WaabiWorld」で実走行の大半を代替、2億ドル調達
   - 数億マイルの仮想走行で数百億マイル実走行に相当するエッジケース訓練が可能（ランド研究所2016年研究）
   - InvertedAI（カナダ）が人間行動予測モデルでNPC（ノンプレイアブル・キャラクター）生成、リアルな訓練環境構築
   - 従来は公道走行距離の積み上げが主流も、仮想環境活用でコスト削減と安全性向上の両立実現

3. **日本発LLM技術「Heron」による文化対応自動運転**
   - チューリング社が日本独自の大規模言語モデル「Heron」開発（最大70億パラメータ）
   - 文章+画像入力に対し文脈理解した自然な文章生成、日本の交通文化に対応
   - 「東京30分ドライブプロジェクト」で都内公道をハンドル介入なしで走行目指す（2025年目標）
   - S.RIDEとの協業でタクシー走行データ収集、日本の自動車産業の層の厚さを活用

4. **マルチモーダルAIによる車内体験の進化**
   - 生成AIの音声認識技術で運転者の声による車両制御が実用化段階
   - 画像認識+音声認識+自然言語処理の統合で、搭乗者とのコミュニケーション高度化
   - 日産が生成AI車載エージェント「AutoDJ」発表（2025年10月）、パーソナライズ化推進
   - 総務省は生成AIによるルート最適化と音声指示の両立を情報通信白書で言及

5. **AI開発の課題と日本の人材不足危機**
   - 経産省推計で2025年にSDV開発人材3.3万人不足、2030年には5.1万人不足に拡大
   - トヨタグループが「トヨタソフトウェアアカデミー」設立、グループ5社共同で人材育成加速
   - AIセキュリティ脆弱性が重大課題、サイバー攻撃による制御不能リスクへのフェイルセーフ必須
   - 責任所在の法整備遅れ、日本では「運行供用者（所有者）」責任が基本も完全自動運転時代に未対応

## 日本企業の先進事例

### チューリング社 - 日本独自LLM「Heron」とレベル5完全自動運転への挑戦

**概要**
- 「We Overtake Tesla」をミッションに2030年完全自動運転EV 10,000台量産目標
- マルチモーダルAIライブラリ「Heron」を世界で初めて最大70億パラメータで開発
- 日本語対応と日本の交通文化理解を重視、米中依存からの脱却狙う
- 2025年に100台納車完了、2027年に量産開始、2023年千葉県柏市に自社工場設立

**技術的特徴**
- **Heron**: LLMに「視覚」を与える学習フレームワーク、文章+画像入力で文脈理解した自然言語生成
- **日本の画像・テキストデータセット**: 大規模な日本独自データで学習、日本の交通文化対応
- **エンド・ツー・エンド手法**: 認知から操作までをAIが一貫処理、ロングテールの事象に対応

**東京30分ドライブプロジェクト**
- 東京都内公道をハンドル介入なしで30分走行する自動運転プロジェクト（2025年目標）
- 信号、交差点、狭い道、路駐車両、走行中自転車、歩行者など複雑要素への対応
- S.RIDEとの協業でタクシーアプリから走行データ収集
- LLMだけでなく物理的走行における人間操作・行動情報も重視

**創業者の危機感**
- 米国カーネギーメロン大学で「日本の自動車メーカーの市場を自動運転で奪う」発言に衝撃
- 日本のITソフトウェアの弱さ（Windows、iOS等）が自動車業界にも波及する危機感
- 「日本の自動車メーカーが軒並みソフトウェア企業の下請けになる未来を変えたい」

## グローバルスタンダード事例

### Tesla（テスラ） - E2E自動運転のパイオニア

**概要**
- 2023年FSD（Full Self-Driving） V.12からE2Eモデルの実装開始
- 従来のモジュール式からセンサー入力を直接処理して運転判断を下す方式へ転換
- イーロン・マスクCEOが早くからレベル5を視野に開発推進
- 現在ADASながらE2E手法の実証で業界をリード

**技術的ブレークスルー**
- **ルールレス学習**: コードに書き込まれた固定ルールではなく、大量の走行データからAIが自律的に運転法則を学習
- **ニューラルネットワーク直接制御**: センサー入力→運転操作を中間システムなしで直結
- **継続的OTA更新**: 週次でソフトウェア更新、全車両の走行データをクラウドで統合学習

**課題と限界**
- 現状はレベル2+のADAS、完全自動運転（レベル5）実現時期は不透明
- 悪天候・複雑交通状況での誤認識リスクが残存
- 法規制・責任所在の不明確さが普及の障壁

## 日本の立ち位置・強み・弱み分析

### 日本の現状と立ち位置

**グローバル市場での位置づけ**
- 自動運転レベル5実現競争で米国（テスラ、Waymo）、中国（百度、XPeng）に完全に後れを取る
- 日本はレベル3市販開始も、レベル4商用サービスは実証実験段階に留まる
- 生成AI×自動運転のE2E手法で米中企業が先行、日本は従来型ルールベースに固執
- AI人材不足が致命的、2030年に5.1万人不足で技術開発競争から脱落の危機

**技術開発体制の現状**
- トヨタグループがソフトウェアアカデミー設立も、グループ内育成では追いつかない
- チューリング社がHeron開発で独自路線も、資金力・人材で米中に劣勢
- 日産がAutoDJ発表も、中国のファーウェイAITO等と比較して実用化遅延
- 仮想環境シミュレーション活用はWaabi、InvertedAI等の海外企業が先行

### 強み（4つの要点）

1. **品質管理・安全検証の深い知見**
   - トヨタ生産方式に代表される品質管理文化は世界最高水準
   - 実車での膨大なテスト走行（数百万km）による安全性検証プロセスが確立
   - APQP/PPAPなど自動車業界標準の品質保証手法を世界に先駆け確立
   - AIの不確実性（確率的動作）と車載環境の高信頼性要求の両立に知見あり

2. **製造業・モノづくりの伝統的強み**
   - 自動車産業の層の厚さ（サプライヤー、部品メーカー、技術者集積）
   - チューリング社が「日本でしかできないチャレンジ」と評価する産学連携基盤
   - 高精度センサー（カメラ、LiDAR、レーダー）の製造技術で世界トップクラス
   - ハードウェア品質の高さがソフトウェアの信頼性を下支え

3. **日本の交通文化への深い理解**
   - チューリング社Heronが日本独自のLLMで日本の交通文化対応
   - 狭い道、複雑な交差点、歩行者・自転車との共存など日本特有の環境対応力
   - 東京30分ドライブプロジェクトで都内公道の複雑要素を克服目指す
   - 日本語対応の自然言語処理で搭乗者とのコミュニケーション最適化

4. **法規制・倫理問題への慎重なアプローチ**
   - トロッコ問題（AI倫理の究極ジレンマ）に対する慎重な姿勢
   - 2020年改正道路交通法・道路運送車両法でレベル3公道走行を世界に先駆け認可
   - 運行供用者（所有者）責任を基本とする法体系で責任所在を明確化
   - フェイルセーフ設計思想が根付き、AI脆弱性への対策重視

### 弱み（4つの要点）

1. **E2E手法への決定的な遅れ**
   - テスラFSD V.12（2023年）、中国XPengがE2E実装も日本企業は未対応
   - 従来のルールベース（人間がルール定義）に固執、パラダイムシフトに適応不能
   - 「すり合わせ型」開発文化がE2E手法の「自律学習」と根本的に不適合
   - 生成AI×自動運転の波に完全に乗り遅れ、技術ギャップ拡大中

2. **仮想環境シミュレーション活用の遅れ**
   - Waabi WaabiWorldが2億ドル調達も、日本では同等技術開発進まず
   - 実走行重視の伝統が仮想環境活用への転換を阻害
   - エッジケース（レアシナリオ）訓練の効率化で米加企業に完全敗北
   - 開発コスト削減の機会を逃し、資本効率で劣勢

3. **AI人材の絶対的不足**
   - 経産省推計で2025年3.3万人、2030年5.1万人不足は致命的
   - トヨタグループがソフトウェアアカデミー設立も、育成速度が需要に追いつかず
   - IT企業との給与格差で優秀人材確保が構造的に不可能
   - 米中が数千人規模のAIエンジニアを擁するも、日本企業は数百人規模に留まる

4. **商用サービス実用化の遅延**
   - 米国Waymo、中国百度Apollo Goがレベル4商用サービス展開も、日本は実証実験段階
   - 法規制の慎重さが実用化スピードを阻害（安全性重視の裏返し）
   - ロボタクシーの収益化で米中に5年以上の遅れ
   - 消費者への完全自動運転体験提供で後手に回り、市場形成で劣勢
