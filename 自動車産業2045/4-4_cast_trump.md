# CASTトレンド分析 4-4 Trump（トランプ政権の影響）

## Summary - 5つの要点

1. **トランプ関税の甚大な影響**
   - 2025年4月から米国輸入乗用車に25%追加関税、5月から自動車部品にも同率賦課
   - 日本には「相互関税」15%に加え自動車関連25%で実質40%の関税負担（2025年8月時点）
   - 日本自動車メーカー大手7社の営業利益に2兆6,000億円超のマイナス影響見込み（2026年3月期）
   - 従来2.5%から27.5%への急激な引き上げ後、日米交渉で15%に緩和も依然高水準

2. **環境規制の劇的緩和とパリ協定離脱**
   - パリ協定から2度目の離脱宣言、温室効果ガス「危険性認定」撤回方針
   - 企業別平均燃費（CAFE）規制の罰金制度廃止
   - カリフォルニア州の「2035年までにガソリン車・HEV販売終了」規制を無効化
   - 北米産EV購入時の最大7,500ドル税額控除が2025年9月末で打ち切り

3. **日本HEVメーカーへの追い風**
   - 「EVだけが脱炭素の道ではない」日本メーカーの立場が政策的に正当化
   - EV化投資・燃費規制罰金が15%関税よりも大きなコスト負担だったが全て消滅
   - 異常なスピードのEV開発圧力が緩和、余裕ある投資計画が可能に
   - 競争力の根源であるエンジン技術を引き続き活用できる4年間の猶予獲得

4. **サプライチェーン再編の加速**
   - メキシコへの25%関税により、メキシコ活用を進めてきた日本企業の北米戦略見直し必須
   - 日産・ホンダが日本生産の一部車種を米国に移管、バイアメリカン政策強化
   - 移民政策厳格化で米国内不法移民の強制送還規模が焦点、人手不足深刻化リスク
   - グローバル調達体制の効率性低下で、消費者価格上昇と競争力低下の二重苦

5. **EV市場の成長鈍化と日本の巻き返し機会**
   - IEA予測で世界EV販売成長率が2023-2025年22.9%から2025-2030年14.4%へ減速
   - トランプ政権4年間は「EVシフトは不可逆に進行するが速度は緩やかになる」（ADL見解）
   - EV購入者への税控除は撤廃まではいかないが絞られる可能性（IRA電池産業補助は継続見込み）
   - 成長鈍化こそ日本の巻き返しチャンス、官民連携で対応が急務

## 日本企業への影響と対応事例

### トヨタ自動車 - 関税コスト自社負担で米国生産・雇用維持

**概要**
- 2025年4月のトランプ関税導入後も、当面は米国での値上げせず
- 関税による上昇分を自社負担し、生産・雇用維持を最優先
- 米国内生産比率が日本メーカー中最高（約70%）で関税影響を最小化
- HEV主力戦略がトランプ環境規制緩和で追い風に

**戦略的対応**
- 世界販売台数1位（2025年3月期売上高48兆円）の体力で関税コスト吸収
- 米国ケンタッキー・テキサス等の工場フル稼働で現地生産最大化
- RAV4等の主力車種を米国生産に集約、日本からの輸出を最小限に
- HEV技術優位性を活かし、EV義務化緩和の恩恵を最大限享受

**中長期リスク**
- 4年後の政権交代で環境規制が再強化される可能性
- 中国・欧州市場でのEV競争では依然劣勢
- トランプ政権終了後の揺り戻しへの備えが課題

**リンク**
- [中日新聞 - トヨタ米で値上げせず関税対応](https://www.chunichi.co.jp/article/901234)

### 日産自動車・ホンダ - 日本生産の米国移管

**概要**
- 日産・ホンダが日本生産の一部車種の生産を米国に移管方針
- トランプ関税コスト回避のため、生産場所変更を断行
- 日産は福岡県苅田町の城下町経済に深刻な影響、減産で地域雇用不安高まる

**日産の苦境**
- 経営危機で2万人の人員削減、工場閉鎖のリストラ実施中
- 米国事業の収益依存度が高く、関税影響が業績悪化に拍車
- ホンダとの経営統合協議打ち切り（2025年2月）で独自戦略継続

**ホンダの対応**
- 日本から「シビック」等を米国に移管
- 二輪車メーカーとしては世界最大、四輪でもグローバル展開
- 飛行機・ロケット製造など多角化でリスク分散

**地方経済への波及**
- 福岡県苅田町では日産工場が地域雇用の主要源、関税影響で不安高まる
- 群馬県太田市のスバル工場など、自動車関連工場が地域経済の柱の都市多数
- 輸出減少は工場稼働率低下→地域経済衰退の連鎖リスク

**リンク**
- [日刊工業新聞 - ホンダシビック米国移管](https://www.nikkan.co.jp/)
- [読売新聞 - 日産苅田町減産の不安](https://www.yomiuri.co.jp/)

### ニッパツ（バネ部品大手） - 米国減産計画を撤回

**概要**
- 人件費高騰を受けて米国での減産を計画していたが、トランプ関税で中断
- 米国での一部部品生産を日本に移管予定だったが、元に戻す方針に転換
- 自動車メーカーが米国生産強化→米国産部品需要高まると判断

**中小部品メーカーへの影響**
- JETRO調査で「米国国内での現地生産増加（輸出からの切り替え）」が28.8%（輸送用機械器具459社）
- 自動車メーカーが米国生産移管→部品メーカーも追随せざるを得ない構図
- 関税コストで自動車メーカー業績悪化→部品メーカーへのコスト削減圧力高まる懸念

**リンク**
- [日本経済新聞 - ニッパツ米国部品減産撤回](https://www.nikkei.com/article/DGXZQOUC160000S5A410C2000000/)
- [Frontier Eyes - トランプ関税と中小部品メーカー](https://frontier-eyes.online/automobile_tariff/)

## グローバル事例と影響分析

### 韓国・ドイツとの相対的競争力変化

**概要**
- 米国市場で日本のライバルである韓国、ドイツも同様に15%相互関税で合意
- 韓国は米韓FTAで2025年3月まで無関税だったため、相対的影響は日本より大
- ドイツは関税影響を受けない現地生産比率が日本より小さく、相対的に不利

**日本の相対的優位性**
- 韓国・ドイツと比較し、日本の競争優位性が高まったと評価可能
- 日本車シェアは上昇見込み（みずほ銀行レポート）
- 米メーカー主力がピックアップトラック・大型SUV、日本は中型SUV・コンパクトカーで市場棲み分け

**リンク**
- [公明党 - トランプ関税の日本自動車産業への影響](https://www.komei.or.jp/km/yachiyo-takahashi-hideyuki/2025/08/25/)

### 米国完成車メーカーの苦境

**概要**
- トランプ関税は米国メーカーも直撃、メキシコ・カナダからの輸入部品に25%課税
- フォード、GMなどが2023年時点でGHG排出基準未達、クレジット購入コスト増大
- 業界全体でクレジット減少、需給逼迫で取引価格上昇の悪循環

**GHG排出基準の厳格化圧力**
- バイデン政権下で2027-2032MY適用の最終規則策定済み
- 2032年に85g/mi目標（2026MY比約50%削減）
- トランプ政権がCAFE罰金制度廃止も、EPA（環境保護庁）の是正権限・リコール権限は残存
- カリフォルニア州ACCII規制（2035年CV販売100%義務化）の行方が焦点

**リンク**
- [JETRO - トランプ米政権と自動車環境規制](https://www.jetro.go.jp/biz/areareports/special/2025/0102/b807011b0dc45648.html)

### カリフォルニア州の独自規制継続

**概要**
- 大気浄化法（CAA）209条でカリフォルニア州は連邦基準より厳しい独自規制が可能
- 2022年制定のACCII規制で2026MY 35%、2035MY 100%のCV（クリーンビークル）販売義務化
- トランプ政権が州権限（Waiver）撤廃を試みるも、法的正当性に疑問

**カリフォルニア州の対抗措置**
- 連邦政府がIRA税額控除撤廃した場合、州独自の7,500ドル控除を復活すると発表
- 2023年9月に財源枯渇で中断していた控除制度の再開準備
- 全米最大の自動車市場（年間200万台規模）での独自規制は業界に無視できない影響

**リンク**
- [JETRO - カリフォルニア州独自規制](https://www.jetro.go.jp/biz/areareports/special/2025/0102/b807011b0dc45648.html)

### 中国BYDへの影響

**概要**
- 米国が2024年9月に「中国部品・ソフトウェア使用コネクテッドカーの輸入・販売禁止」
- 安全保障上のリスクに加え、自国自動車産業保護の狙い
- トランプ関税でBYDの米国市場参入は事実上不可能に

**BYDの戦略転換**
- 米国市場は断念、東南アジア・欧州・中南米・中東へグローバル展開加速
- メキシコ工場計画は凍結、タイ・インドネシア・ブラジルでの生産拠点構築に注力
- 日本市場でも2025年内に年間3万台販売目標、トランプ政策の逆風を他市場でカバー

**リンク**
- [みずほ銀行 - トランプ2.0の北米自動車産業への影響](https://www.mizuhobank.co.jp/corporate/industry/pdf/msif_240.pdf)

4. **サプライチェーン再編の不可避性**

   - 移民政策厳格化で米国内の人手不足深刻化、現地生産拡大の足枷に

   - トランプ政権4年間は「EVシフト緩和」「HEV優遇」の黄金期
   - 2029年の政権交代で環境規制が再強化される可能性が極めて高い
   - 中国・欧州は関係なくEV化推進、日本だけが米国市場に最適化する戦略リスク
   - 短期的追い風に甘えず、EV競争力強化の手を緩めるべきではない

## 日本の立ち位置・強み・弱み分析

### 日本の現状と立ち位置

**グローバル市場での位置づけ**
- 米国市場が日本自動車メーカー最大の輸出先、トランプ関税の直撃を受ける
- 中国市場でEV競争に敗北しつつあるなか、米国市場での優位性が生命線
- トランプ環境規制緩和でHEV優位性が短期的に高まるも、長期的には不透明
- 4年間の猶予期間をEV競争力強化に活用できるかが日本の命運を左右

**政策対応の現状**
- 日本政府が緊急経済対策、公共料金補助策で対応も焼け石に水
- 日米二国間交渉で関税15%に緩和も、25%自動車追加関税は継続
- 官民連携でEV巻き返しを図るも、中国勢との技術ギャップ拡大中

### 強み（4つの要点）

1. **HEV技術の圧倒的優位性**
   - トヨタTHSは20年以上の実績、燃費50-60km/Lは世界最高水準
   - トランプ環境規制緩和で「EVだけが脱炭素の道ではない」主張が政策的に正当化
   - 充電インフラ不要で即座に環境性能向上できる実用性が再評価
   - エンジン技術を引き続き活用でき、4年間の開発余裕獲得

2. **米国現地生産比率の高さ**
   - トヨタは米国生産比率約70%で関税影響を最小化
   - ケンタッキー・テキサス等の工場で年間200万台規模の現地生産体制
   - 日系メーカー全体で米国雇用15万人超を創出、政治的プレゼンス確保
   - バイアメリカン政策への適合度が韓国・ドイツより高い

3. **中型SUV・コンパクトカーでの市場棲み分け**
   - 米メーカー主力がピックアップトラック・大型SUV、日本は中型SUV・コンパクト
   - 市場セグメントが異なり、正面衝突を回避
   - 燃費効率重視の顧客層に日本車の優位性、トランプ政権下でも需要は堅調
   - RAV4、CR-V、カムリ等の主力車種は米国生産で関税回避

4. **品質管理文化とブランド力**
   - トヨタ、ホンダの信頼性・耐久性は米国消費者に深く浸透
   - リコール率の低さ、長期保有価値の高さが差別化要因
   - 急速な生産拠点移管でも品質維持へのコミットメントが競争力の源泉
   - 米国市場での日本車シェアは20-30%を維持、ブランドロイヤリティ高い

### 弱み（4つの要点）

1. **EV競争での決定的後れ**
   - トランプ政権4年間の「EV緩和」に甘えると、中国勢との差がさらに拡大
   - 2029年の政権交代で環境規制再強化されると完全に取り残される危険性
   - 欧州・中国市場では関係なくEV化推進、米国市場のみに最適化は戦略的誤り
   - 全固体電池実用化（トヨタ2027-28年目標）が遅延すればEV競争から脱落

2. **関税コスト吸収の限界**
   - トヨタは自社負担で乗り切る体力あるも、日産・ホンダ・マツダ・スバルは厳しい
   - 日本大手7社で営業利益2兆6,000億円減は業界全体の利益の20-30%に相当
   - 関税コストを価格転嫁すれば販売減少、自社負担すれば業績悪化の二者択一
   - 中小部品メーカーへのコスト削減圧力が限界を超え、品質低下・倒産リスク

3. **米国現地生産急拡大のリスク**
   - 日産・ホンダが急遽生産移管→米国工場の急拡大は品質管理体制の急造を意味
   - 日本国内の「すり合わせ型」品質管理が米国工場で再現困難
   - 熟練工不足、サプライヤー連携の弱さが品質低下・リコール増加のリスク
   - 移民政策厳格化で米国内の人手不足深刻化、現地生産拡大が困難に

4. **地方経済への深刻な波及**
   - 日本国内工場の稼働率低下→地域雇用減少→地方経済衰退の連鎖
   - 福岡県苅田町（日産）、群馬県太田市（スバル）など自動車城下町の危機
   - 中小部品メーカーの倒産・撤退が地域産業集積の崩壊を招く
   - 自動車産業が地域の主要雇用源の都市多数、社会問題化の懸念
