# なぜ今、超小型原子炉が「最も旨味のある技術」なのか
## 市場機会分析レポート 2025

**作成日**: 2025年10月16日  
**分析者**: 品質保証エンジニア（投資・市場分析視点）

---

## 🎯 エグゼクティブサマリー

**結論**: 服部禎男氏が1980年代に提唱した超小型原子炉は、当時は「旨味がない」とされたが、**2025年現在、最も利益率が高く、成長性が高い技術分野**に変貌した。

### なぜ今なのか？

| 要因 | 1980年代 | 2025年 | 変化 |
|------|---------|--------|------|
| **データセンター需要** | 存在せず | **爆発的成長** | AI革命 |
| **市場規模** | 不明確 | **$4,500億（2040年）** | 明確化 |
| **電力価格** | 低価格（$20-30/MWh） | **高価格（$60-200/MWh）** | 3-7倍 |
| **カーボン価格** | ゼロ | **$50-100/tCO2** | 脱炭素政策 |
| **投資環境** | 政府依存 | **VC・民間資金豊富** | 金融革命 |
| **規制環境** | 硬直的 | **柔軟化（米国）** | 規制改革 |
| **技術実証** | なし | **EBR-II等の実績** | データ蓄積 |
| **競合** | 大型原発・火力 | **再エネ＋蓄電池（不安定）** | 優位性明確 |

**利益率予測**:
- **粗利益率**: 40-60%（製造業平均の2-3倍）
- **営業利益率**: 25-40%（従来原発の3-5倍）
- **ROIC**: 20-35%（投資効率極めて高）

#### なぜ超小型炉が最適か

**データセンターの電力要件**:
```
✓ 24時間365日安定供給（稼働率>99%）
✓ オンサイト発電（送電ロス回避）
✓ カーボンフリー（ESG要求）
✓ 迅速な建設（2-3年）
✓ スケーラブル（段階的増設）
✓ 用地効率（コンパクト）

→ 超小型炉がすべて満たす
→ 再エネ＋蓄電池では不可能
```
