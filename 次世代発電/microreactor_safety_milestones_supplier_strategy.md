### 🛡️ TRISO燃料：究極の安全設計

#### 多重バリア構造（Fail-Safe by Design）

```
TRISO粒子の層構造（直径~1mm）:

  役割: 核分裂ガス膨張吸収

  ↓ 第3の封じ込め（最重要）

  ↓ 第4の封じ込め

→ 合計4層の独立バリア

### 🔥 メルトダウン不可能性の物理的根拠

#### 温度限界の比較

```
従来軽水炉:
• 燃料融点: UO₂ 2,800°C
• 被覆管融点: Zr合金 1,850°C
• 通常運転: 320°C
• 事故時到達: >2,000°C（メルトダウン）
→ 事故時に融点超過リスク

TRISO燃料マイクロ炉:
• SiC健全性維持: <1,600°C
• 通常運転: 500-750°C
• 事故時最高: <1,500°C（受動冷却）
• 安全マージン: >100°C
→ 物理的にメルトダウン不可能
```

#### 受動安全メカニズム

**1. ヒートパイプ冷却（Westinghouse eVinci™）**
```
原理: ナトリウムの自然循環
• 高温部でNa蒸発 → 熱吸収
• 蒸気が低温部へ移動
• 凝縮して熱放出
• 液体が毛細管力で戻る
→ ポンプ・電源不要

利点:
✓ 自己制御（温度上昇→冷却増）
✓ 機械的可動部なし（MTBF極大）
✓ 全電源喪失でも機能
✓ 宇宙実証済み技術（50年実績）
```

**2. 自然循環冷却（三菱重工）**
```
原理: 密度差駆動
• 炉心加熱 → 冷却材密度低下
• 浮力で上昇
• 熱交換器で冷却 → 密度増
• 重力で下降
→ ポンプ不要

条件:
• 適切な高さ差（2-5m）
• 低圧系（<1MPa）
• 十分な熱交換面積

→ Fukushima型事故完全回避
```

**3. 負の反応度フィードバック**
```
温度上昇時:
1. ドップラー効果
   → U-238中性子吸収増
   → 核分裂率低下
   
2. 熱膨張
   → 燃料間隔拡大
   → 中性子漏洩増
   → 反応度低下
   
3. 冷却材密度低下
   → 減速能低下
   → 反応度低下

→ 自動的に出力抑制（<1秒）
→ 制御棒なしでも安全
```
