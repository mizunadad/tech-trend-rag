# 技術トレンド相談システム（家族用） - Streamlit版

## 目的
本システムは、Gartnerや日経BPの技術レポートに基づき、家族それぞれのキャリアや興味に合わせた技術トレンド情報を提供し、学習や進路のサポートを行うことを目的としています。

## 使い方

1.  <strong>Streamlit Webアプリ</strong>にアクセスします。（URLはデプロイ後に発行されます）
2.  <strong>ユーザー名とパスワード</strong>を入力してログインします。（家族ごとの認証が必要です）
3.  以下の質問例を参考に、個別の進路相談や技術調査を行ってください。

### 質問例

**大学生（化学系）の息子さん向け**:
免疫について知見を広めたいです。将来の技術を視野に入れて役に立ちそうな技術を教えてください。

**中学生（スイマー）の息子さん向け**:
そろそろ水中で生活しようと思うようになりました。将来、水中で生活できるようになる未来はやってくるのでしょうか。

**かわいい奥様向け**:
レモンがたくさんなりました。美味しくいただけるレシピを紹介してください。また、思い浮かべるだけで料理ができる将来はやってくるのでしょうか。

**キャリア相談（開発者向け）**
半導体デバイスの知識を元に、もっと生活に役立つ楽しい世界創りに貢献したいです。そのためにはどのような技術トレンドに沿って研鑽すればいいでしょうか。
