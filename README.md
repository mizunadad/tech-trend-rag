# 🧬 NEXT-GEN CAREER BRAIN
### 技術トレンド活用型キャリア探索・分析システム

## 📌 概要 (Overview)
本システムは、Gartnerや日経BP等の最新技術トレンド（約900件）をナレッジベースとし、ユーザーのキャリアや興味関心を将来の市場ニーズと照らし合わせて探索するためのRAG（検索拡張生成）アプリケーションです。

単なる技術検索にとどまらず、**「具体⇔抽象」の思考プロセス**をAIが補助し、多角的な視点から将来のキャリアパスや研究テーマを検討することを目的としています。

---

## 📊 データセットと前処理 (Data Preparation)
信頼性の高い技術情報を収集し、分析に適した形式で構造化・データベース化しています。

* **データソース**: Gartnerハイプ・サイクル、日経BP技術レポート等
* **データ処理**:
    * Markdown形式による均一フォーマット化（Obsidianスタイル準拠）
    * メタデータ付与（Tags, 日本の強み・弱み, 国内外事例）
    * Pythonスクリプトによるベクトル化およびFirestoreへの格納

---

## ⚡ 主な機能 (Core Features)

### 1. RAG検索・分析
ユーザーの質問に対して、データベースから関連性の高い技術情報を抽出し、LLMが回答を生成します。回答の根拠となる資料のタイトルおよび原文コンテキストも確認可能です。

### 2. 思考の深掘り・拡張 (Knowledge Elevator)
検索された技術トピックに対し、視点を切り替えた分析を提供します。
* **⬆️ 抽象化 (Abstract)**: 個別の事例から、共通する成功要因や構造的な強みを抽出。
* **⬇️ 具体化 (Concrete)**: 抽出された強みを活かした、2030年の具体的な社会実装プランを提案。
* **↔️ 横展開 (Analogous)**: 技術スタックが異なる他分野への転用や、イノベーションの機会を示唆。

### 3. 将来シミュレーション (Simulation)
技術が普及した未来を具体的にイメージするためのシナリオ生成機能です。
* **未来の名刺**: その技術領域の専門家となった場合の、2035年時点での職業プロフィールとスキルセットを予測。
* **未来の日記**: その技術が日常化した世界での生活風景をシミュレーションし、ショートショート形式で出力。

## 💻 開発ワークフロー (Development Workflow)
本システムは以下のプロセスを経て構築されました。

1.  **データパイプライン**: Python (`sentence-transformers`) を用いたMarkdownファイルの解析・ベクトル化と、NoSQLデータベース (Firestore) への格納。
2.  **アーキテクチャ選定**: 認証・DB接続・LLM連携をPythonのみで完結させるため、Streamlit Cloudを採用。
3.  **UI/UX実装**:
    * 家族ごとのログイン認証機能の実装
    * Graphviz/Mermaidによるシステム構成図および技術体系マップの可視化
    * プロンプトエンジニアリングによる、日本語出力精度の調整
