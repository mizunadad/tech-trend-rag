# 技術トレンド相談システム（家族用）

## 使い方

1. Claude.aiにアクセス: https://claude.ai/
2. Projectセクションから「Tech Trend RAG」を選択
3. 以下のような質問をしてください：

### 質問例

**大学生（化学系）の息子さん向け**:
 免疫について知見を広めたいです。将来の技術を視野に入れて
役に立ちそうな技術を教えてください。

**中学生（スイマー）の息子さん向け**:
 そろそろ水中で生活しようと思うようになりました。
でも水中ではスマホを使うことができません。将来、水中で
生活できるようになる未来はやってくるのでしょうか。

**かわいい奥様向け**:
 レモンがたくさんなりました。美味しくいただけるレシピを
紹介してください。また、思い浮かべるだけで料理ができる
将来はやってくるのでしょうか。

**キャリア相談**
 半導体デバイスの知識を元に、もっと生活に役立つ楽しい
世界創りに貢献したいです。そのためにはどのような技術
トレンドに沿って研鑽すればいいでしょうか。
