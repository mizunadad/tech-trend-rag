# T9-09-01 道路埋設型送電コイル・インフラ整備

## Summary

高速道路や幹線道路の車線下に電磁誘導コイルを数メートル間隔で埋設し、走行中のEVトラック・バスに連続的にワイヤレス給電する革新的インフラ技術。韓国KAIST(OLEV)が世界初の実証、スウェーデンeRoadArlandaが2km区間で実用化、日本は東京大学・国土交通省が柏市で公道実証実験を2023年開始。建設コストは1km=2-10億円と高額だが、長距離トラック・バスのバッテリー容量を1/5に削減でき、車両コスト大幅低減と航続距離無制限化を実現。2025年大阪万博での実証、2030年代の幹線道路への本格導入を目指す。物流脱炭素化の切り札として、トラック・バス専用レーンから優先整備する戦略が世界的トレンド。

### 5つの技術要点

1. **電磁誘導コイル埋設方式**: 道路下5-10cm深さに送電コイルを数メートル間隔で配置、車両の受電コイルと磁界共振結合により85-90%の高効率で電力伝送
2. **セクション制御・自動給電システム**: 車両接近時のみコイルを通電、通過後自動停止するスマート制御で安全性確保と電力損失最小化
3. **高出力給電技術**: 大型トラック向け100kW級、バス向け50-100kW、乗用車向け20kWの可変出力対応により多様な車種に対応
4. **耐久性・メンテナンス**: 道路舗装材と一体化したコイル構造で20年以上の耐用年数、振動・温度変化・塩害に耐える設計
5. **段階的整備戦略**: 固定ルートの路線バス・長距離トラック専用レーンから優先整備し、投資効果を最大化しながら段階的に拡大

## 具体的プロダクト事例

### 日本企業の先進事例

**東京大学 + 国土交通省 柏の葉実証実験**
- 概要: 2023年10月開始の国内初公道走行中給電実証
- 技術: 磁界共振結合方式、時速100km走行で20kW給電
- URL: https://www.jst.go.jp/pr/jst-news/
- 特徴: 柏市柏の葉キャンパス地区で継続中、2025年大阪万博へ技術移転

**大成建設 + 大成ロテック 走行中給電実証**
- 概要: 時速60kmで最大10kW連続給電を実証
- 技術: 20m区間送電コイル埋設、伝送効率66-71%達成
- URL: https://www.taisei.co.jp/about_us/wn/2025/250718_10570.html
- 特徴: 舗装テストトラックで高速走行試験、コスト低減型設計

**デンソー + 豊田自動織機 受電システム開発**
- 概要: 車両側受電コイル・制御システム開発
- 技術: 高速移動対応・位置ずれ自動補正
- 特徴: トヨタグループの電動化戦略の中核技術

**東武バスウエスト 川越路線実証(環境省GX基金)**
- 概要: 2025年営業路線での長期運用開始
- 技術: 停車中給電(SWPT)から走行中給電(DWPT)へ展開
- 特徴: 本格導入を見据えた世界初の営業運用実証

**NEDO万博プロジェクト(2025)**
- 概要: 大阪・関西万博会場の来場者移動用EVバスに実装
- 技術: グリーンイノベーション基金での大規模実証
- 特徴: 国際的なショーケースとして技術発信

### グローバルスタンダード

**韓国KAIST OLEV(Online Electric Vehicle)**
- 概要: 2009年世界初の走行中給電シャトルバス実証
- 技術: 2013年Gumi市で24km路線、144m給電区間運用
- URL: https://www.electricroads.org/technologies/
- 課題: 老朽化インフラで2016年までに4路線全て停止、2019年Yuseong地区で再始動も商業化に苦戦

**スウェーデン eRoadArlanda**
- 概要: 2017-2019年運用の世界初実用電化道路
- 技術: Elways社の地中導電レール方式、2km区間、200kW給電
- URL: https://eroadarlanda.com
- 特徴: Arlanda空港-物流拠点間、使用量自動計測・課金システム実装、塩水浸水時も1V以下の安全設計

**スウェーデン E20高速道路電化計画(Trafikverket)**
- 概要: 2023年開始、21km区間の永久電化道路建設
- 技術: Stockholm-Gothenburg-Malmö間、2035年まで3000km拡大計画
- URL: https://www.trafikverket.se
- 現状: 2024年12月最終報告で費用対効果不足と判断、計画一時停止

**イスラエル Electreon(旧ElectRoad)**
- 概要: 都市部電動バス向け動的ワイヤレス給電
- 技術: 2021年Gotland島1.65km、Tel Aviv 700m実証
- URL: https://electreon.com
- 展開: 米国Detroit 2023年0.25マイル区間(16kW給電)、Michigan州での展開拡大中

**米国 ASPIRE(Utah State University)**
- 概要: NSF 2670万ドル資金でワイヤレス給電研究センター
- 技術: 2015年SELECT実証から発展、2020年ASPIRE設立
- 実績: Florida州Orlando近郊SR516に1マイル・200kW給電システム(1300万ドル)建設中
- URL: https://aspire.usu.edu

**中国 Zhongtong電気化公路**
- 概要: 2026年までに500km電化道路建設計画
- 技術: 新疆地区炭鉱トラック向けパンタグラフ式架線給電
- 特徴: 24時間連続運用、完全電動化物流システム

**ノルウェー Trondheim AtB 動的誘導充電**
- 概要: 2024-2025年バス路線実証
- 技術: 80m誘導コイル区間、時速30kmで120kW送電・65kW受電
- 特徴: 北欧の寒冷地での動作検証

## 全体要約の特徴

道路埋設型送電コイル技術は、走行中のEVに連続給電することで航続距離無制限化を実現する革新的インフラ。韓国KAISTが2009年世界初実証、スウェーデンが2017年2km実用化、日本は2023年公道実証開始と段階的に進展。建設コスト1km=2-10億円と高額だが、大型トラックのバッテリー容量を1/5に削減でき、車両コスト大幅低減を実現。2025年大阪万博でのショーケース後、固定ルート路線バス・長距離トラック専用レーンから優先整備する戦略が世界的コンセンサス。

技術的には磁界共振結合方式が主流(効率85-90%)だが、パンタグラフ式架線給電(中国)、導電レール方式(スウェーデンElways)など複数方式が競合中。国際標準規格は未確立で、2021年からWiPowerOne・Electreon・IPT等が標準化活動推進。安全性は実証済み(塩水浸水時1V以下)だが、外来異物検知・電磁波人体影響評価は継続課題。

経済性では、1km整備に2-10億円の初期投資が必要だが、売電収入(45円/kWh想定)で長期回収可能との試算(KPMG)。新東名高速道路の2/3電化で792億円投資、年間数十億円売電収入見込み。トラック運送業の燃料費削減、車両軽量化による積載効率向上等の間接効果も大きい。

## 日本の立ち位置・競争力分析

### 強み(Strengths)

1. **基礎研究力と技術開発**: 東京大学・名古屋大学が磁界共振結合の基礎研究で世界トップクラス、藤本博志教授チームが柏市で国内初公道実証成功。大成建設・デンソー・豊田自動織機等の製造業が高効率受電システム・耐久性コイル設計で技術蓄積。
2. **品質管理・長寿命設計**: 道路舗装技術(大成ロテック)と電子デバイス信頼性評価(半導体品質保証)の融合により、20年以上の耐用年数確保。振動・温度変化・塩害対応の日本独自ノウハウ。
3. **政府主導の実証プロジェクト**: 環境省GX基金、NEDO万博プロジェクト、国土交通省ガイドライン策定(2024)等、官民連携の推進体制整備。EVワイヤレス給電協議会(WEV)発足で業界横断の標準化活動開始。
4. **自動車産業エコシステム**: トヨタ・ホンダ・日産等の完成車メーカー、デンソー・パナソニック等のティア1サプライヤー、半導体メーカー(ローム・三菱電機)の垂直統合による迅速な実装能力。

### 弱み・課題(Weaknesses)

1. **実用化・商業化の遅延**: 韓国2009年、スウェーデン2017年実用化に対し、日本は2023年公道実証開始と10-15年遅延。2025年万博実証が本格導入への試金石だが、海外に先行されリーダーシップ喪失リスク。
2. **建設コスト高**: 日本の道路建設費は欧米の1.5-2倍、1km=5-10億円と試算され、スウェーデン(1-2億円)比で3-5倍高額。公共事業予算制約下での大規模インフラ投資の実現性に疑問。
3. **規格統一・国際標準化の遅れ**: 磁界共振結合・導電レール・パンタグラフ式等、複数方式が並存し国際標準未確立。日本独自規格推進のリスク(ガラパゴス化)と、海外規格追従のジレンマ。SAE/IEC標準化議論への影響力不足。
4. **スタートアップ・ビジネスモデル不在**: 韓国WiPowerOne、イスラエルElectreon、米国ASPIRE等のベンチャー主導に対し、日本は大企業・大学中心で機動性欠如。課金システム・エネルギーマネジメント等のソフトウェア・ビジネスモデル開発が弱い。
