# T1-11-05 自動点検・補修ロボット

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 実用期初期（2025-2028年）
**技術成熟度**: TRL 6-8（実証から初期展開段階）

## Summary（5つの要点）

1. **3K作業（きつい・危険・汚い）からの解放**: 橋梁・トンネルの点検・補修は、高所作業、狭隘空間、交通規制下での作業が多く、作業員の安全リスクが高い。高齢化・人手不足が進む中、危険作業をロボットに代替する技術が不可欠。国交省は「建設現場の生産性向上」を目標に、2025年までに点検・補修ロボットの現場実装を推進。

2. **多様なロボット技術の統合**: ①橋梁下面点検ロボット（磁力走行、ワイヤー懸垂）、②トンネル内走行ロボット（レーザースキャナ、画像撮影）、③配管内点検ロボット（管内自走）、④壁面補修ロボット（ひび割れ注入、塗装）、⑤水中点検ロボット（ROV: 遠隔操作型無人潜水機）。各ロボットは、自律走行、遠隔操作、AI画像診断、センサー統合、アーム作業等の技術を組み合わせて構成。

3. **AI画像診断との連携**: ロボット搭載カメラで撮影した画像をAIが即座に解析し、損傷箇所を自動検出。検出結果をロボットにフィードバックし、損傷箇所に自動接近・詳細撮影。人間の技術者は遠隔地から画像を確認し、補修要否を判断。点検時間を従来の1/3-1/5に短縮し、交通規制期間を最小化。

4. **自律制御と遠隔操作のハイブリッド**: 完全自律走行が理想だが、複雑な構造物では遠隔操作が必要。5G/Beyond 5G通信により、低遅延（1ms以下）で高精細映像を伝送し、遠隔地から直感的なロボット操作が可能。VR/ARグラスで作業員が「ロボットの目」を通して現場を見ながら操作。自律制御（障害物回避、経路計画）と遠隔操作を適切に組み合わせることで、安全性と作業効率を両立。

5. **補修作業の自動化**: 点検だけでなく、補修作業もロボット化。①ひび割れ注入ロボット（エポキシ樹脂自動注入）、②塗装ロボット（防錆塗料スプレー）、③ボルト締結ロボット（トルク管理）、④溶接ロボット（鋼橋の補修溶接）。建設機械メーカー（コマツ、日立建機）、産業ロボットメーカー（ファナック、安川電機）が技術開発を加速。2030年には、「点検から補修まで無人化」が実現する見込み。

## 具体的プロダクト事例

### 日本企業の先進事例

#### 1. **首都高速道路「橋梁点検ロボット（磁力走行型）」**
- **URL**: https://www.shutoko.co.jp/
- **概要**: 首都高速道路公社が開発した磁力走行式の橋梁点検ロボット。高所作業車不要で点検コスト削減。
- **技術特徴**:
  - 強力磁石で鋼橋下面に吸着し、自律走行
  - 4Kカメラ+AI画像診断でひび割れ・錆びを自動検出
  - 遠隔操作（5G通信）とAI自律制御のハイブリッド
  - バッテリー駆動（連続3時間動作）
- **成果**: 点検時間を70%削減、高所作業の安全性向上、年間コスト1億円削減

#### 2. **川崎重工業「トンネル点検ロボット」**
- **URL**: https://www.khi.co.jp/
- **概要**: レーザースキャナ搭載の自走式トンネル点検ロボット。覆工コンクリートの変状を高速計測。
- **技術特徴**:
  - 3Dレーザースキャナ（精度±1mm）でトンネル全断面を計測
  - 高解像度カメラ（8K）でひび割れ・漏水を撮影
  - SLAM（自己位置推定）により自律走行
  - AI画像診断で損傷箇所を自動分類
- **成果**: 従来の目視点検（2週間）を2日に短縮、交通規制不要

#### 3. **ロボデックス「壁面補修ロボット（ひび割れ注入）」**
- **URL**: https://www.robotdex.jp/
- **概要**: 壁面を吸着走行し、ひび割れに自動でエポキシ樹脂を注入する補修ロボット。
- **技術特徴**:
  - 真空吸着パッド（負圧）で壁面に吸着
  - ひび割れ検出センサー（画像認識）で注入箇所を自動判定
  - 高圧注入ノズル（圧力0.3MPa）で樹脂を深部まで充填
  - 遠隔操作（Wi-Fi）で作業監視
- **導入実績**: トンネル覆工、橋脚、ビル外壁の補修

#### 4. **日本電気（NEC）「水中点検ロボット（ROV）」**
- **URL**: https://jpn.nec.com/
- **概要**: ダム・港湾構造物の水中部を点検するROV（遠隔操作型無人潜水機）。
- **技術特徴**:
  - 水深100mまで潜航可能
  - 高感度カメラ（水中用LED照明）でコンクリート表面を撮影
  - 超音波厚さ計でコンクリート厚を非破壊測定
  - マニピュレータ（ロボットアーム）で水中サンプル採取
- **導入実績**: 国交省のダム管理、港湾局の岸壁点検

#### 5. **安川電機「産業用ロボットアームによる溶接補修」**
- **URL**: https://www.yaskawa.co.jp/
- **概要**: 産業ロボット世界4大メーカーの一角。溶接ロボットを橋梁補修に応用。
- **技術特徴**:
  - 6軸多関節ロボットアームで複雑形状に対応
  - レーザーセンサーで溶接線を自動認識
  - アーク溶接、スポット溶接、レーザー溶接に対応
  - 遠隔監視（5G通信）で熟練技術者が指導
- **導入実績**: 鋼橋の亀裂補修、造船所のドック作業

### グローバルスタンダード

#### 1. **Boston Dynamics（米国）「Spot（4脚歩行ロボット）」**
- **URL**: https://www.bostondynamics.com/
- **概要**: 世界的に有名な4脚歩行ロボット。階段・不整地を自在に移動し、インフラ点検に活用。
- **技術特徴**:
  - 360度カメラ、LIDAR、深度センサーで周囲を認識
  - 自律歩行（障害物回避、経路計画）
  - アタッチメント交換（カメラ、ガス検知器、マニピュレータ）
  - 遠隔操作（タブレット）で簡単制御
- **導入実績**: 米国の発電所、工場、建設現場、日本の建設会社（鹿島建設、大成建設）

#### 2. **Flyability（スイス）「Elios 3（衝突耐性ドローン）」**
- **URL**: https://www.flyability.com/
- **概要**: 狭隘空間・暗所を飛行可能な衝突耐性ドローン。トンネル・橋梁内部の点検に最適。
- **技術特徴**:
  - 球形ケージで壁面衝突しても墜落しない
  - LIDAR（3Dマッピング）+ 4Kカメラで暗所撮影
  - 自律飛行モード（GPS不要、ビジュアルSLAM）
  - 点群データとAI診断ソフトを統合
- **導入実績**: 欧州のトンネル点検、米国の橋梁内部点検、日本の高速道路会社

#### 3. **ULC Robotics（米国）「ガス管・水道管点検ロボット」**
- **URL**: https://www.ulcrobotics.com/
- **概要**: 配管内を自走し、腐食・クラックを検出するロボット。世界最大のガス管点検ロボットメーカー。
- **技術特徴**:
  - 直径150mm〜1200mmの管に対応
  - 360度カメラ + 超音波センサーで管壁厚を測定
  - AIで腐食レベルを5段階判定
  - リアルタイム映像伝送（5G）
- **導入実績**: 米国のガス会社、日本の東京ガス・大阪ガス

#### 4. **SMP Robotics（ロシア）「自律警備ロボット」**
- **URL**: https://smprobotics.com/
- **概要**: 完全自律走行の警備ロボット。インフラ施設（ダム、発電所）の巡回点検に応用。
- **技術特徴**:
  - GPS + LIDAR + カメラで自律走行
  - 異常検知（侵入者、火災、ガス漏れ）を自動通報
  - 夜間監視（赤外線カメラ）
  - 無線充電ステーションで自動充電
- **導入実績**: 欧州の発電所、空港、工場

#### 5. **CUES（米国）「管路点検ロボット（下水道・水道）」**
- **URL**: https://www.cuesinc.com/
- **概要**: 下水道管・水道管の点検ロボットで世界シェアトップ。管内TVカメラ調査の標準機。
- **技術特徴**:
  - クローラー走行（直径200mm〜3000mm対応）
  - 高解像度カメラ（4K）+ LED照明
  - 管内状況を自動記録（動画+位置情報）
  - AI画像診断で損傷箇所を自動検出
- **導入実績**: 全世界の自治体上下水道局、日本の下水道管理者

## 全体要約（5つの要点）

### 1. 技術の本質
自動点検・補修ロボットは、「人間が立ち入れない・立ち入りたくない危険箇所での作業を代替し、安全性と効率性を両立する技術」である。高所、狭隘空間、水中、有毒ガス環境等での点検・補修作業を自律走行・遠隔操作ロボットが実施。AI画像診断、センサー統合、マニピュレータ制御により、従来の人手作業と同等以上の精度を実現。人手不足・高齢化が進む日本の建設業界にとって、生き残りの鍵となる技術。

### 2. コア技術要素
- **移動技術**: クローラー走行、磁力吸着、真空吸着、4脚歩行、ドローン飛行、水中推進
- **自律制御**: SLAM（自己位置推定）、障害物回避、経路計画、AI判断
- **遠隔操作**: 5G/Beyond 5G通信、VR/AR可視化、ハプティクスフィードバック
- **センサー**: カメラ（4K/8K）、LIDAR、超音波、赤外線、ガス検知器
- **作業機能**: マニピュレータ（ロボットアーム）、注入ノズル、溶接トーチ、スプレーガン

### 3. 応用分野
- **橋梁点検**: 鋼橋下面の磁力走行点検、コンクリート橋の外観点検
- **トンネル点検**: 覆工コンクリートのレーザースキャン、漏水・ひび割れ検出
- **ダム・港湾**: 水中点検ROV、堤体変位測定
- **配管点検**: 上下水道管、ガス管の内部点検
- **補修作業**: ひび割れ注入、防錆塗装、溶接、ボルト締結

### 4. 主要プレイヤー
- **日本**: 首都高速道路、川崎重工業、NEC、安川電機、ファナック、コマツ、日立建機、鹿島建設、大成建設
- **米国**: Boston Dynamics、ULC Robotics、CUES、Honeybee Robotics
- **欧州**: Flyability（スイス）、SMP Robotics（ロシア）、ANYbotics（スイス）
- **中国**: DJI（ドローン）、UBTECH Robotics

### 5. 将来展望
2025-2030年にかけて、国交省の「建設現場生産性向上」政策により、点検・補修ロボットの導入が加速。Beyond 5G通信（低遅延1ms以下、超高速10Gbps）の整備により、遠隔操作の精度が飛躍的に向上。AI技術の進化により、完全自律点検（人間の監視不要）が実現。2030年以降、補修作業の自動化（ひび割れ注入、塗装、溶接）が本格化し、「点検から補修まで無人化」が達成される見込み。ロボット技術と建設技術の融合により、「建設ロボティクス」という新産業分野が確立する。

## 日本の立ち位置（4つの要点）

### 1. 強み：産業ロボット技術の蓄積

日本は産業用ロボット（製造ライン）で世界トップシェア（ファナック、安川電機、川崎重工業、不二越）を持ち、ロボットアーム、制御技術、センサー統合のノウハウが豊富。これらの技術をインフラ点検・補修に応用することで、高精度・高信頼性のロボットを開発できる。

また、日本は地震・台風等の災害経験から、建設ロボット（無人化施工）の技術が発展。雲仙普賢岳噴火災害（1991年）、東日本大震災（2011年）での無人化施工の実績があり、遠隔操作技術、自律制御技術が高度に発達している。

### 2. 弱み：コスト高と現場の保守性

日本のロボット技術は高性能だが、コストが高い（欧米の2-3倍、中国の5-10倍）。自治体・中小建設会社は予算制約が厳しく、高価なロボットを導入しにくい。レンタル・リースモデル、RaaS（Robotics as a Service）の普及が必要だが、ビジネスモデルの確立が遅れている。

また、建設業界は保守的で、新技術の導入に慎重。「ロボットが見落とした損傷で事故が起きたら誰が責任を取るのか」という懸念があり、最終的に人間の技術者が再確認する運用が一般的。ロボットの信頼性向上と、法的責任の明確化（ロボット点検の法的位置づけ）が課題。

### 3. 機会：国内需要拡大とアジア展開

日本国内では、2030年までに建設業就業者が現在の500万人から400万人以下に減少する見込み。人手不足を補うため、点検・補修ロボットの需要が急拡大。国交省は「新技術活用システム（NETIS）」でロボット技術を評価・認定しており、公共工事での採用が進む。

アジア諸国（ASEAN、インド）でもインフラ老朽化が進んでおり、日本のロボット技術への期待が大きい。特に、地震・台風等の災害リスクが高い地域では、日本の無人化施工技術が有効。日本政府の「質の高いインフラ輸出」戦略と連動し、ロボット技術の海外展開チャンスがある。

### 4. 脅威：欧米のプラットフォーム支配と中国の低価格攻勢

米国のBoston Dynamics「Spot」、スイスのFlyability「Elios」は、グローバル市場で事実上の標準となりつつある。日本企業がこれらのプラットフォーム上でアプリケーションを開発する「下請け」的な立場に甘んじるリスクがある。

中国企業（DJI等）は、低価格ドローンで世界市場を席巻しており、点検ロボット分野でも同様の戦略を展開する可能性。日本企業は、高品質・高信頼性を武器に差別化する必要があるが、コスト競争力の向上も不可欠。

また、ロボット点検の法的基準が未整備であり、「ロボット点検だけで近接目視点検を省略できるか」が曖昧。国交省は段階的に規制緩和を進めているが、完全な無人化には至っていない。法整備の遅れが、技術開発のスピードにブレーキをかけている。
