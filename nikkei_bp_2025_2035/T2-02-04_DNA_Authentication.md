# T2-02-04 DNA鑑定・品種真正性検証技術

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 成長期初期（2026-2029年）
**技術成熟度**: TRL 6-7（実証実験から初期商用展開段階）

## Summary（5つの要点）

1. **科学的産地・品種証明**: 魚介類、肉類、米、野菜の品種・産地をDNA解析で科学的に証明し、産地偽装を防止。高級食材（マグロ、和牛、有機野菜）の真正性保証に不可欠。PCR法、次世代シーケンサーによる高精度判定。

2. **ポータブルDNAシーケンサーによる現場判定**: Oxford Nanopore MinION等の携帯型次世代シーケンサーにより、現場（港湾、市場、レストラン）で数時間以内にDNA鑑定が可能。従来の実験室分析（数日〜数週間）から迅速判定への転換。

3. **多様な食材への対応**: 米品種（コシヒカリ、あきたこまち等）、肉種（牛、豚、鶏、馬）、魚種（マグロ、サーモン、ウナギ）、野菜（大豆、小豆、白インゲン豆）の品種識別。加工食品（レトルト、冷凍）でもDNA断片から同定可能。

4. **環境保全型農業製品の認証**: 有機農産物、特別栽培農産物の品種・産地証明。JAS有機認証、GAP認証の科学的根拠を提供。遺伝子組換え（GMO）作物の混入検査（ISO 21570準拠）。

5. **国際貿易での信頼性担保**: 輸出入食品の品種・産地証明により、国際貿易での信頼性を確保。EU、米国、中国等の輸入規制対応。違法漁獲（IUU漁業）対策、絶滅危惧種保護。

## 具体的プロダクト事例

### 日本企業の先進事例

#### 1. **日本食品分析センター（JFRL） DNA品種識別検査サービス**
- **URL**: https://www.jfrl.or.jp/
- **概要**: ISO/IEC 17025認定試験所として、米・肉・魚・野菜のDNA品種識別検査を提供。PCR法、リアルタイムPCR法を用いた高精度判定。
- **成果**: コシヒカリ新潟BL品種判別、国産ウナギ識別等で実績

#### 2. **ビジョンバイオ DNA鑑定サービス**
- **URL**: https://visionbio.com/
- **概要**: 米品種判別、肉種DNA判別、魚介類DNA判別、動植物DNA同定検査を提供。加工食品（レトルト、冷凍）にも対応。
- **成果**: 食品メーカー、小売チェーンでの品質管理に活用

#### 3. **MERIEUX NUTRISCIENCES JAPAN DNA検査**
- **URL**: https://bv-foodtesting.jp/
- **概要**: 遺伝子組換え体検査、作物品種識別検査、動物由来物質含有検査を提供。EU・米国の国際基準対応。
- **成果**: 輸出入食品のGMO検査で高シェア

#### 4. **バイオロジカ Eurofins MediGenomix提携DNA検査**
- **URL**: https://www.biologica.co.jp/
- **概要**: ドイツEurofins MediGenomix社と提携し、食品表示のDNA解析を実施。牛肉の雌雄判定、内容物生物種同定、遺伝子改変判定。
- **成果**: 高級食材の真正性証明に活用

#### 5. **ハウス食品分析テクノサービス FRED法異物検査**
- **URL**: https://food-analab.jp/
- **概要**: DNA分解程度を測定するFRED法により、食品混入クロゴキブリの加熱履歴を評価。混入時期・製造時加熱有無を推定。
- **成果**: 農研機構と共同開発、食品異物混入原因究明に貢献

### グローバルスタンダード

#### 1. **Oxford Nanopore MinIONポータブルシーケンサー（英国）**
- **URL**: https://nanoporetech.com/
- **概要**: USB接続の超小型次世代シーケンサー（重量87g）。現場でDNA鑑定を数時間で実施可能。
- **導入実績**: WHO、FDA、FAO等の食品検査機関で採用

#### 2. **Illumina MiSeq次世代シーケンサー（米国）**
- **URL**: https://www.illumina.com/
- **概要**: 世界最大の次世代シーケンサーメーカー。食品DNA検査用ベンチトップ型シーケンサー。
- **導入実績**: グローバル食品検査機関の標準機器

#### 3. **Thermo Fisher Scientific遺伝子解析システム（米国）**
- **URL**: https://www.thermofisher.com/
- **概要**: リアルタイムPCR装置、DNAシーケンサー、試薬キットを包括提供。食品GMO検査の国際標準。
- **導入実績**: FDA、EFSA、各国食品検査機関で採用

#### 4. **QIAGEN食品DNA抽出キット（ドイツ）**
- **URL**: https://www.qiagen.com/
- **概要**: 加工食品からのDNA抽出キット。高温処理・冷凍食品でも高純度DNA抽出が可能。
- **導入実績**: 世界中の食品検査ラボで標準キットとして使用

#### 5. **Eurofins食品DNA検査サービス（欧州）**
- **URL**: https://www.eurofins.com/
- **概要**: 世界最大の食品検査機関ネットワーク。DNA品種識別、産地証明、GMO検査を包括提供。
- **導入実績**: 欧州・米国・アジアで広範な検査網を展開
