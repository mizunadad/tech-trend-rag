## Summary（5つの要点）

1. **航続距離無制限のEV社会**: 道路に埋設したコイルから走行中の車両に電力を供給し、航続距離を事実上無制限にする革命的技術。大容量バッテリー搭載が不要となり、車両重量30-50%削減、車両価格20-30%低減を実現可能

2. **2025年大阪万博で世界初の大規模実証**: 関西電力、Osaka Metro、大林組、ダイヘン、NEXCO東日本が万博会場外周道路100mにDWPTシステムを設置。EVバス6台で走行中給電を実証し、日本の技術力を世界に発信

3. **段階的社会実装ロードマップ**: トラック・バス路線から優先整備(2025-2030年)、高速道路追い越し車線への拡大(2030-2035年)、一般道への展開(2035年以降)。建設コストを1km=10億円から5億円以下へ削減が課題

4. **再生可能エネルギー統合**: 太陽光発電の余剰電力を日中の走行中給電で活用し、ダックカーブ現象(夜間一斉充電による電力需要急増)を回避。EVを移動する分散蓄電池として活用し、電力系統安定化に貢献

5. **日本の技術開発**: 東京大学・名古屋大学が基礎研究をリード、ホンダが150kW級高出力システムを開発中。ダイヘンが大阪万博で15kW×2コイル=30kW給電を実証。韓国KAIST、スウェーデンeRoadArlandaが先行実用化

## 具体的プロダクト事例

### 日本企業の先進事例

**関西電力・Osaka Metro・大林組・ダイヘン・NEXCO東日本コンソーシアム**
- **プロジェクト**: 2025年大阪・関西万博 走行中ワイヤレス給電(DWPT)実証
- **設置場所**: 万博会場外周道路100m区間(西端直線道路)+ バス停道路
- **技術仕様**: ダイヘン製送電コイル20個(プレキャストコンクリート内蔵)、出力15kW/コイル×2=30kW/車両
- **対象車両**: EVバス6台(会場周回バス約30台中)
- **運用期間**: 2025年4月13日~10月13日(万博開催期間)
- **認可**: 2025年4月18日に総務省から高周波利用設備の認可取得(電磁波安全性確認済)
- **特徴**: 世界初の大規模DWPT実証、走行速度20-40km/h対応、再エネ統合EMS構築
- **リンク**: [EXPO 2025 EVバス](https://www.expo2025.or.jp/future-index/smart-mobility/evbus/)

**東京大学大学院 新領域創成科学研究科 藤本・清水研究室**
- **プロジェクト**: 柏の葉キャンパス駅周辺公道でのDWPT実証(日本初)
- **期間**: 2023年10月~2025年3月
- **技術仕様**: 送電コイル1つあたり25kW、2コイル以上で合計50kW受電
- **パートナー**: ブリヂストン、デンソー、国土交通省公募事業
- **実証内容**: 公道での走行中給電、車両動特性への影響評価、効率測定
- **特徴**: 大学発の基礎研究を公道実証まで推進、データ公開による標準化貢献
- **リンク**: [東京大学 藤本研究室](https://www.k.u-tokyo.ac.jp/pros/person/hiroaki_fujimoto/hiroaki_fujimoto.htm)

**ホンダ(本田技術研究所)**
- **製品**: 150kW級高出力DWPTシステム(開発中)
- **技術仕様**: 送電コイル1つあたり最大150kW(大型車・乗用車共用設計)
- **実証車**: ホンダNSX改造車、車両側ユニット寸法550×550mm、厚さ22mm
- **特徴**: 乗用車と大型車の給電システム共用化により導入コスト削減を実現
- **開発戦略**: トラック・バス路線から普及開始、乗用車への展開を見据えた高出力設計
- **並行開発**: パンタグラフ式有線給電システムも同時開発(用途に応じて使い分け)
- **リンク**: [日経クロステック - ホンダDWPT記事](https://xtech.nikkei.com/atcl/nxt/column/18/00001/09666/)

**ダイヘン株式会社**
- **製品**: 走行中ワイヤレス給電システム(コイル式・平行二線式)
- **技術仕様**: コイル式15-50kW(EV・バス・トラック向け)、平行二線式~10kW(低コスト型)
- **実績**: 2025年大阪万博で給電装置を提供、大林組が道路施工を担当
- **開発方針**: 高出力化とコスト削減の両立、自動物流道路への適用検討
- **協議会**: EVワイヤレス給電協議会幹事会社(100超の団体参加)
- **リンク**: [ダイヘン 走行中ワイヤレス給電](https://www.daihen.co.jp/products/wireless/dwpt/)

**NEXCO東日本(東日本高速道路)**
- **プロジェクト**: 大阪万博実証成果を活用した高速道路へのDWPT実装研究
- **対象**: 新東名高速道路、東名高速道路での導入検討
- **戦略**: 追い越し車線への優先整備、長距離トラック・バスの燃料費削減
- **課題**: 建設コスト1km=5-10億円の削減、メンテナンス体制構築
- **目標**: 2030年代の一部区間での試験導入、2040年代の本格展開

### グローバルスタンダード

**韓国KAIST(Korea Advanced Institute of Science and Technology)**
- **プロジェクト**: OLEV(Online Electric Vehicle)システム - 世界初の実用化
- **技術仕様**: 磁界共鳴方式、送電コイル出力20kW/個、伝送距離20cm、効率85%
- **実績**: 2013年ソウル市内バス路線で商用運行開始(世界初)、2016年龜尾市バス路線で拡大
- **走行距離**: 合計24km区間にDWPTシステム設置、バス12台運行
- **技術革新**: 道路埋設コイルの薄型化(5cm以下)、車高変化への対応技術確立
- **リンク**: [KAIST OLEV Project](https://www.kaist.ac.kr/)

**スウェーデン eRoadArlanda**
- **プロジェクト**: 世界初の商用トラック向けDWPT道路
- **場所**: ストックホルム・アーランダ空港連絡道路2km区間
- **技術仕様**: 電力レール方式(平行二線式)、出力50-200kW、効率90%以上
- **運用開始**: 2018年から商用運行開始、電動トラックが空港貨物輸送に使用
- **特徴**: 道路埋設コストを削減する電力レール方式採用、積雪・凍結環境での実績
- **展開**: 2025年までにストックホルム近郊3,000km区間への拡大計画
- **リンク**: [eRoadArlanda公式サイト](https://www.eroadarlanda.com/)

**イスラエル ElectReon**
- **設立**: 2013年、イスラエル発のDWPTスタートアップ
- **技術**: 磁界共鳴方式、道路埋設コイル、出力20-200kW、効率85-90%
- **実績**: イスラエル・テルアビブで電動バス路線実証(2020年~)
- **国際展開**: スウェーデン・ゴットランド島(2km区間)、ドイツ・バウテン(1km区間)、イタリア・ブレシア
- **パートナー**: Stellantis(フィアット・クライスラー)、EnBW(ドイツ電力)
- **特徴**: モジュール式道路埋設システムで施工期間短縮、後付け改修が容易
- **リンク**: [ElectReon公式サイト](https://www.electreon.com/)

**中国 Yutong Bus(宇通客車) × WiTricity**
- **プロジェクト**: 自動運転eバスへのワイヤレス充電商用化(世界初)
- **発表**: 2023年2月、10人乗り自動運転ミニバスにWiTricity Haloソリューション導入
- **技術仕様**: 磁界共鳴方式、出力11kW(停車中)、将来的にDWPT対応予定
- **特徴**: 自動運転×ワイヤレス充電の完全無人運用、フリート管理システム統合
- **展開**: 中国国内の空港・工業団地で展開、2025年以降の量産計画
- **リンク**: [Yutong Bus公式サイト](https://www.yutong.com/)

**Momentum Dynamics(米国)**
- **技術**: 超高出力磁界共鳴式DWPT(商用車特化)
- **技術仕様**: 出力75-300kW、伝送距離15-30cm、効率90-93%
- **実績**: ペンシルベニア州フィラデルフィアで電動バス200kW充電実証
- **戦略**: バス・トラック向け超高出力充電に特化、デポ・バスターミナルでの停車中充電から展開
- **開発方針**: 停車中充電(SWPT)で実績蓄積後、走行中充電(DWPT)へ拡大
- **リンク**: [Momentum Dynamics](https://www.momentumdynamics.com/)

## My Notes

**品質保証エンジニアの視点からの技術評価**
- **道路埋設コイルの耐久性**: 道路に埋設される送電コイルは、車両荷重(大型トラック20トン超)、温度変化(-20℃~+60℃)、水分浸入、塩害に耐える必要がある。プレキャストコンクリート内蔵型の防水性能IP68、振動試験(IEC 60068-2-64)が必須
- **車両側受電コイルの位置検出精度**: 時速40-100km走行中に送電コイル上を通過する際、受電コイルとの位置合わせを高速で行う必要がある。GPS+IMU+磁気センサーによる位置検出精度±5cm以内を保証
- **電磁波漏洩の安全性評価**: 走行中給電は人が近接する環境で使用されるため、電磁波漏洩(CISPR 11)と人体影響(ICNIRP基準)の厳格な評価が必要。総務省認可取得が社会実装の前提条件
- **給電効率の速度依存性**: 走行速度に応じて送受電コイルの結合係数が変化し、効率が低下。速度20km/h(効率85%)、60km/h(効率70%)、100km/h(効率50%)の速度特性評価とワーストケース保証

**Pythonデータ解析への応用**
```python
# 走行中給電の効率・速度特性モデル
import numpy as np
import matplotlib.pyplot as plt

def dwpt_efficiency(speed_kmh, base_efficiency=85):
    """走行速度に対する給電効率モデル"""
    # 速度上昇に伴うコイル結合時間減少による効率低下
    efficiency = base_efficiency * np.exp(-0.015 * speed_kmh)
    return efficiency

speeds = np.linspace(0, 120, 100)
efficiencies = dwpt_efficiency(speeds)

plt.plot(speeds, efficiencies)
plt.xlabel('Vehicle Speed (km/h)')
plt.ylabel('Power Transfer Efficiency (%)')
plt.title('DWPT Efficiency vs Speed Characteristics')
plt.grid(True)
plt.axhline(y=70, color='r', linestyle='--', label='Target Efficiency (70%)')
plt.legend()
plt.show()

# バッテリー容量削減効果のシミュレーション
# DWPT導入によるLCCO2削減量の定量評価
def battery_reduction(route_distance_km, dwpt_coverage_ratio):
    """DWPT導入によるバッテリー容量削減率"""
    # DWPTカバー率50%で、必要バッテリー容量を60%削減可能
    reduction_ratio = 0.6 * dwpt_coverage_ratio
    reduced_battery = 100 - (100 * reduction_ratio)  # kWh
    return reduced_battery

coverage_ratios = np.linspace(0, 1, 50)
battery_capacities = battery_reduction(300, coverage_ratios)

plt.plot(coverage_ratios * 100, battery_capacities)
plt.xlabel('DWPT Coverage Ratio (%)')
plt.ylabel('Required Battery Capacity (kWh)')
plt.title('Battery Capacity Reduction by DWPT Coverage')
plt.grid(True)
plt.show()
```

## 全体要約の特徴(5つの要点)

1. **EV普及の最大障壁を解消**: 走行中給電により航続距離を事実上無制限化し、充電時間ゼロ、大容量バッテリー不要(車両重量30-50%削減)、車両価格20-30%低減を実現。EV普及の最大障壁である充電インフラ問題を根本解決

2. **2025年大阪万博で世界初の大規模実証**: 関西電力・Osaka Metro・大林組・ダイヘン・NEXCO東日本コンソーシアムが万博会場外周100mにDWPTシステム設置、EVバス6台で走行中給電を実証。日本の技術力を世界に発信し、国際標準化をリード

3. **段階的社会実装ロードマップ**: トラック・バス路線から優先整備(2025-2030年、建設コスト回収しやすい)→高速道路追い越し車線(2030-2035年)→一般道(2035年以降)。建設コスト1km=10億円を5億円以下へ削減が実用化の鍵

4. **再エネ統合とカーボンニュートラル**: 太陽光余剰電力を日中の走行中給電で活用し、ダックカーブ現象(夜間一斉充電)回避。EVを移動する分散蓄電池として活用し、電力系統安定化に貢献。運輸部門CO2排出量18.5%を大幅削減

5. **グローバル競争の本格化**: 韓国KAIST(2013年世界初実用化)、スウェーデンeRoadArlanda(2018年商用運行)、イスラエルElectReonが先行。日本は東大・名大の基礎研究とホンダ150kW級システム開発で巻き返し、2025年万博で技術力を実証

## 日本の立ち位置と強み・弱み（4つの要点）

### 強み

1. **産官学連携の推進体制**: 関西電力・Osaka Metro・大林組・ダイヘン・NEXCO東日本コンソーシアムが2025年大阪万博で世界初の大規模実証を実施。国土交通省、経済産業省、総務省が規制緩和・標準化・補助金で支援

2. **土木・建設技術の優位性**: 大林組が道路埋設コイルの施工技術を確立、プレキャストコンクリート工法により施工期間短縮。日本の高品質な道路インフラ維持管理技術をDWPTに応用し、長寿命化・低コスト化を実現

3. **高効率パワーエレクトロニクス**: ダイヘンが85kHz高周波インバーター技術で世界トップレベル。SiC/GaNパワー半導体の活用により、給電効率90%以上を達成。日本の半導体技術がDWPTの心臓部を支える

4. **自動車メーカーの積極参入**: ホンダが150kW級高出力DWPTシステムを開発中、トヨタ・日産もWiTricityと技術提携。自動車産業の総合力を活かし、車両側受電システムの最適化と量産化でコスト削減を推進

### 弱み

1. **実用化の遅れ**: 韓国KAIST(2013年実用化)、スウェーデンeRoadArlanda(2018年商用運行)に対し、日本は2025年万博実証が初の大規模展開。商用運行実績で10年以上の遅れ、技術検証データの蓄積不足

2. **建設コストの高さ**: 日本の試算で1km=10億円(韓国・スウェーデンは5-7億円)。道路工事コスト、プレキャストコンクリート製造コスト、労務費が高く、全国展開には7兆円規模の投資が必要で実現性に課題

3. **規制・認可プロセスの複雑さ**: 総務省(電波法)、国土交通省(道路法)、経済産業省(電気事業法)の三重規制により、実証実験から商用化まで5-10年を要する。韓国・中国の迅速な規制緩和と対照的

4. **スタートアップ・ベンチャー不足**: イスラエルElectReon、米国Momentum Dynamicsのような革新的スタートアップが日本に存在せず、大企業・大学主導の保守的開発スタイル。リスクを取った技術革新・事業化推進力が不足
