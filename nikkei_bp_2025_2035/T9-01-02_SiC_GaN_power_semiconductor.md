# T9-01-02 パワーエレクトロニクス・SiC/GaN半導体

## ハイプ・サイクル位置: 期待のピーク〜幻滅期

## 5つの要点Summary

1. **SiC/GaN半導体の性能優位性**: シリコン比10倍の電力損失削減、200℃高温動作、高周波スイッチング対応。EV航続距離10-20%延長、インバータ小型化50%、冷却システム簡略化を実現

2. **8インチウエハー量産競争**: 2025年から8インチSiCウエハーの量産工場が相次ぎ稼働。ローム(宮崎)、三菱電機(熊本)、富士電機(松本)が2025-27年に稼働開始予定。生産効率2倍向上、コスト30%削減目標

3. **EV向け採用拡大**: テスラModel 3が2017年に採用後、日産アリア、レクサスRZ等で標準化。2025年以降投入されるEVの80%以上がSiC採用予測。市場規模は2021年1400億円→2030年3.4兆円(24倍成長)

4. **日本企業の競争力**: SiCパワー半導体世界シェアでローム4位、三菱電機・富士電機がトップ10入り。ウエハーからデバイスまで垂直統合型生産体制を構築。2027年度までに累計5100億円投資(ローム)

5. **GaNの小型機器展開**: スマホ急速充電器、データセンター電源で先行普及。SiC比で小型・高周波だが高コスト。台湾TSMC、韓国Samsungが受託製造強化、水平分業型産業構造が定着

## 全体要約の特徴

### 省エネ社会を支える基盤技術
SiC/GaNパワー半導体は、従来のシリコン半導体の物理的限界を突破し、電力損失を10分の1に削減する革新技術である。EV用インバータに採用されることで航続距離が10-20%延長し、太陽光発電のパワーコンディショナーでは変換効率が98%超に達する。テスラModel 3が2017年にSiCインバータを採用して以降、EV業界のデファクトスタンダードとなり、2025年以降発売されるEVの80%以上がSiC搭載予測。

### 8インチウエハー量産が競争の焦点
生産コスト削減と供給能力拡大の鍵を握るのが8インチウエハーへの移行。ウエハー面積が6インチ比2.25倍となり、1枚あたりのチップ取得数が大幅増加。2025年はローム、三菱電機、Infineon等が相次ぎ8インチ工場を稼働させる「SiC量産元年」。日本勢は8インチ対応で欧米と同時期だが、工場自動化・生産効率で追いつく必要がある。

### 垂直統合 vs 水平分業の産業構造
SiCは垂直統合型、GaNは水平分業型という対照的な産業構造が形成されつつある。SiCはウエハー製造が技術的に難しく、ローム(独SiCrystal買収)、三菱電機(米Coherent出資)等がウエハーからデバイスまで一貫生産体制を構築。一方GaNはファブレス企業(Navitas、GaN Systems)がTSMCに製造委託する水平分業モデルが主流。日本はSiCで優位だが、GaN水平分業への対応が課題。

### 用途の棲み分けと市場展望
SiCは600V以上の高耐圧用途(EV、鉄道、太陽光発電、UPS)で優位。GaNは小型・高周波が求められる用途(スマホ充電器、データセンター電源、5G基地局)で先行普及。市場規模はSiCが2030年に3.4兆円(2021年比24倍)、GaNが2030年に5000億円規模に拡大予測。2050年には次世代パワー半導体がシリコンを上回ると考えられる。

### 日本の技術優位性と課題
日本企業はSiCパワー半導体世界シェアでトップ10に5社(三菱電機、富士電機、ローム、東芝、ルネサス)ランクイン、合計20%超のシェアを保持。60年間のパワー半導体製造ノウハウ、車載品質基準(ISO 26262)対応力、IGBT時代からのパッケージング技術が強み。課題は8インチ量産の遅れ(欧米比2-3年)、GaN水平分業への対応不足、中国勢の低価格攻勢への対抗。

## 日本の立ち位置 - 強み・弱み4点Summary

### 強み

1. **垂直統合型生産体制とサプライチェーン支配力**
   - ローム(独SiCrystal買収)、三菱電機(米Coherent出資)がウエハーからデバイスまで垂直統合を実現

   - ISO 26262機能安全規格対応、AEC-Q101車載グレード認証取得の実績
   - IGBTモジュール時代から蓄積したパッケージング技術、放熱設計、ゲートドライバIC統合技術

3. **パワーモジュール技術とシステム統合力**

   - 半導体産業を経済安全保障の要と位置づけ、国内生産基盤強化を国家戦略として推進

### 弱み・課題

1. **8インチ量産開始の2-3年遅れ**
   - Infineon(2024年稼働)、Wolfspeed(2022年稼働)に対し、日本勢は2025-27年稼働で後手
   - 8インチ対応の製造装置・プロセス開発で欧米半導体装置メーカー(Applied Materials、Lam Research)に依存
   - 後工程(ダイシング、ワイヤボンディング、樹脂封止)の自動化・省人化でも欧米に遅れ

2. **GaN水平分業モデルへの対応不足**
   - GaN市場はファブレス+ファウンドリ(TSMC)の水平分業モデルが主流
   - 日本のGaN受託製造体制が未整備。JSファンダリ(2024年夏稼働)等の新規参入組に期待
   - TSMCの150mm GaN-on-Si製造技術、Samsung Electronicsの200mm対応(2025年)に対抗できる国内ファウンドリ不在

3. **中国勢の政府支援と低価格攻勢**
   - 中国Silan Microelectronics、UNT等が政府支援で8インチSiC工場建設(投資額120億人民元)
   - 中国市場(世界EV販売の60%)を背景に、量産効果で価格を30-50%削減する戦略
   - 技術的には日本より劣るが、コスト競争力で中国・東南アジア市場を席巻するリスク

4. **国際標準化とエコシステム構築の遅れ**
   - AEC-Q規格(車載部品品質規格)、IEC国際標準でのプレゼンス不足
   - 欧米主導の標準化活動(ECPE European Center for Power Electronics等)に日本企業の参画が限定的
   - オープンイノベーション・エコシステム構築で後れ。大学・研究機関・スタートアップとの連携不足

## 品質保証エンジニアの視点

### 信頼性評価の重要項目

#### 1. 高温動作長期信頼性試験(HTOL: High Temperature Operating Life)
**試験条件**
- 温度: 200℃(SiCの強みを活かす極限条件)
- 電圧: 定格電圧の80%印加
- 試験時間: 1000時間以上(車載グレードは2000時間)
- サンプルサイズ: 77個以上(信頼度60%、信頼水準90%)

**評価項目**
- オン抵抗変化率: ±20%以内(AEC-Q101規格)
- リーク電流増加率: 初期値の10倍以下
- 閾値電圧変動: ±10%以内
- 故障モード: ゲート酸化膜劣化、接合部劣化、ワイヤボンド剥離

**データ解析**
- ワイブル解析によるB10寿命(10%故障時の寿命)算出
- アレニウスモデルによる加速係数計算、実使用温度での寿命予測
- 故障物理解析(FIB-SEM、TEM観察)による劣化メカニズム特定

#### 2. 温度サイクル試験(TCT: Temperature Cycle Test)
**試験条件**
- 温度範囲: -55℃〜200℃(SiCの広動作温度範囲)
- サイクル: 1000サイクル(15分保持、昇降温速度5℃/min)
- 評価: 100サイクルごとに電気特性測定、X線検査

**故障モード**
- ワイヤボンド剥離(Al線とSiCチップの熱膨張係数差)
- はんだクラック(ダイアタッチ部、銅ベースプレート接合部)
- パッケージ樹脂クラック(EMC樹脂の熱膨張)
- チップクラック(SiC基板内部の転位・マイクロクラック)

**対策技術**
- 銅ワイヤボンディング(Al線比で高信頼性)
- 焼結銀ダイアタッチ(はんだ比で高温耐性)
- 低熱膨張率樹脂(エポキシモールディングコンパウンド最適化)

#### 3. 短絡耐量試験(SC: Short Circuit)
**試験目的**
- IGBTと比較したSiC-MOSFETの短絡耐量評価
- 過電流保護回路の応答時間余裕確認

**試験条件**
- 電圧: 定格電圧の80%(例: 1200V品→960V)
- 短絡時間: 1μs、3μs、10μs(段階的に延長)
- 繰り返し: 各条件10回、電気特性変化を追跡

**SiCの特性**
- 短絡耐量時間: 3-10μs(IGBT 10μsより短い)
- ジャンクション温度上昇速度: 5000℃/ms(IGBTの2倍高速)
- 保護回路要件: 検出1μs+遮断2μs=合計3μs以内の高速保護必須

**設計対策**
- デサチュレーション検出回路(過電流検知)の高速化
- ゲートドライバICの過電流保護機能統合(1μs以内遮断)
- 並列接続チップ数の最適化(短絡電流分散)

#### 4. EMC評価(電磁両立性)
**高速スイッチングに伴うEMI課題**
- SiC-MOSFETのスイッチング周波数: 100kHz-1MHz(Si-IGBT 20kHz比で10倍)
- dv/dt: 50V/ns超(Si-IGBT 5V/ns比で10倍)
- di/dt: 10A/ns超(高速化で配線インダクタンスとの干渉)

**評価項目**
- 放射エミッション: CISPR 25 Class 5(自動車EMC規格)
- 伝導エミッション: 0.15-108MHz、dBμV基準
- サージイミュニティ: ISO 7637-2(車載電子機器サージ試験)
- ESD耐性: HBM ±2kV、CDM ±500V

**対策技術**
- ゲート抵抗最適化(スイッチング速度とリンギング抑制のトレードオフ)
- PCBレイアウト最適化(低インダクタンス配線、グランドプレーン強化)
- EMIフィルタ設計(コモンモードチョーク、Xコンデンサ配置)
- シールドパッケージ採用(金属ケース、導電性樹脂)

#### 5. 製造工程品質管理

**ウエハー検査**
- 欠陥密度: <1 defect/cm²(6インチウエハーで50個以下)
- 結晶欠陥: マイクロパイプ、転位密度<10⁴ cm⁻²
- 不純物濃度: ドーパント均一性±10%以内
- 検査手法: フォトルミネッセンス(PL)マッピング、X線トポグラフィー

**電気特性全数検査**
- オン抵抗: ±15%以内(ビニング選別で±5%達成)
- 耐圧: 定格の125%で実測(1200V品→1500V印加)
- リーク電流: 定格電圧印加時10μA以下
- ゲート閾値電圧: 4V中心±1V(ゲートドライバ設計余裕)

**工程能力指数**
- Cpk目標: 1.67以上(不良率0.6ppm = 3σ品質)
- 統計的プロセス管理(SPC): X-R管理図による工程監視
- 歩留まり向上: 80%→90%→95%と段階的改善

### Pythonによるデータ解析例

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats
from scipy.optimize import curve_fit
import seaborn as sns

# SiC-MOSFETの高温動作試験(HTOL)データ解析
data = pd.read_csv('SiC_MOSFET_HTOL_data.csv')

# デバイスID、試験時間、オン抵抗、リーク電流、閾値電圧のデータ
# グループ化: 初期(0h)、中間(500h)、最終(1000h)
initial = data[data['test_hours'] == 0]
mid = data[data['test_hours'] == 500]
final = data[data['test_hours'] == 1000]

# オン抵抗の変化率計算
ron_change = ((final['Ron_mohm'].values - initial['Ron_mohm'].values) / 
              initial['Ron_mohm'].values) * 100

# 統計解析
print("=== SiC-MOSFET HTOL Test Results (200°C, 1000h) ===")
print(f"オン抵抗変化率: 平均 {ron_change.mean():.2f}%, 標準偏差 {ron_change.std():.2f}%")
print(f"AEC-Q101規格(±20%以内): {'合格' if abs(ron_change).max() < 20 else '不合格'}")

# リーク電流の増加評価
leak_initial = initial['leak_current_uA'].values
leak_final = final['leak_current_uA'].values
leak_ratio = leak_final / leak_initial

print(f"\nリーク電流増加率: 平均 {leak_ratio.mean():.2f}倍, 最大 {leak_ratio.max():.2f}倍")
print(f"規格(10倍以下): {'合格' if leak_ratio.max() < 10 else '不合格'}")

# ワイブル解析による故障予測
failure_threshold = 30  # オン抵抗変化率30%を故障とみなす
time_to_failure = []

for device_id in data['device_id'].unique():
    device_data = data[data['device_id'] == device_id].sort_values('test_hours')
    ron_initial = device_data[device_data['test_hours'] == 0]['Ron_mohm'].values[0]
    for _, row in device_data.iterrows():
        ron_change_rate = abs((row['Ron_mohm'] - ron_initial) / ron_initial) * 100
        if ron_change_rate > failure_threshold:
            time_to_failure.append(row['test_hours'])
            break

if len(time_to_failure) > 2:

# アレニウスモデルによる加速係数計算
def arrhenius_acceleration_factor(T_use, T_stress, Ea=1.0):
    """
    アレニウスモデルによる加速係数計算
    T_use: 実使用温度(℃)
    T_stress: 加速試験温度(℃)
    Ea: 活性化エネルギー(eV) ※SiC MOSFETは1.0-1.5eV程度
    """
    k = 8.617e-5  # ボルツマン定数(eV/K)
    T_use_K = T_use + 273.15
    T_stress_K = T_stress + 273.15
    
    AF = np.exp((Ea/k) * (1/T_use_K - 1/T_stress_K))
    return AF

# 実使用温度125℃、加速試験200℃の場合
T_use = 125
T_stress = 200
Ea = 1.2  # SiC MOSFETの典型的な活性化エネルギー

AF = arrhenius_acceleration_factor(T_use, T_stress, Ea)
print(f"\n=== 加速係数計算(アレニウスモデル) ===")
print(f"実使用温度: {T_use}℃")
print(f"加速試験温度: {T_stress}℃")
print(f"活性化エネルギー: {Ea} eV")
print(f"加速係数: {AF:.1f}倍")

if len(time_to_failure) > 2:
    # 実使用温度での寿命予測
    b10_life_use = b10_life * AF
    mttf_use = mttf * AF
    print(f"\n実使用温度{T_use}℃でのB10寿命: {int(b10_life_use)}時間 ({int(b10_life_use/8760):.1f}年)")
    print(f"実使用温度{T_use}℃でのMTTF: {int(mttf_use)}時間 ({int(mttf_use/8760):.1f}年)")
```
