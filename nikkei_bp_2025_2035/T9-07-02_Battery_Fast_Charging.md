# T9-07-02 大容量バッテリーパック・急速充電技術

## Summary: 5つの要点

1. **バッテリー大容量化の標準化**: PHEV20kWh、EV60-100kWh以上が標準となり、プレミアムEVでは100kWh超も登場。トヨタ新型プリウスPHEVは13.6kWh、アルファードPHEVは18.1kWhを搭載

2. **CHAdeMO 350kW時代の到来**: e-Mobility Powerと東光高岳が2025年秋から世界最高出力350kWの急速充電器展開開始。10分充電で400km走行を実現し、電圧1000V対応で高効率充電が可能

3. **次世代ChaoJi規格への移行**: 日中共同開発の超高出力規格ChaoJi(最大900kW)が2032-35年に実用化予定。現行CHAdeMO/GB/Tから統一され、アジア市場でのデファクトスタンダード化を目指す

4. **バッテリーコスト半減目標**: 2030年までにkWhあたりコストを半減し、EVの価格競争力を向上。パナソニック・LG化学・CATLが量産技術開発を推進、中国CATLがLFP電池で低コスト化をリード

5. **充電規格の世界分断**: CHAdeMO(日本・アジア)、CCS(欧州・北米)、NACS(Tesla)、GB/T(中国)と4-5規格が乱立。日産が北米で2025年からNACS採用を決定し、CHAdeMO一辺倒から戦略転換

## 技術詳細

### 技術概要

PHEVは20kWh、EVは60-100kWh以上のバッテリーを搭載。急速充電器(CHAdeMO、CCS、Tesla Supercharger)で30分で80%充電を実現。次世代350kW超急速充電により10分以下を目指す。

**主要技術要素**:
- リチウムイオンバッテリー(三元系NMC、LFP)
- バッテリーマネジメントシステム(BMS)
- 温度管理システム(液冷・空冷ハイブリッド)
- 急速充電プロトコル(CHAdeMO、CCS、NACS)
- 充電カーブ最適化(バッテリー保護と充電速度の両立)

### 日本企業の先進事例

**e-Mobility Power + 東光高岳**
- **製品**: CHAdeMO規格で世界最高350kW出力の急速充電器
- **展開時期**: 2025年秋から設置開始
- **技術仕様**: 最大電圧1000V、電流350A、800Vバッテリー搭載EVに最適化
- **充電性能**: 10分で400km走行分を充電(バッテリー容量80kWh想定)
- **設置場所**: 高速道路SA/PA、都市部充電ステーション
- **URL**: https://www.e-mobipower.co.jp/

**CHAdeMO協議会**
- **規格**: 日本発の急速充電規格、2010年策定、2014年国際標準承認
- **設置数**: 国内12,806基(2025年6月)、世界90カ国以上で展開
- **通信方式**: CAN(Controller Area Network)でバッテリー状態を逐次確認
- **出力範囲**: 50kW(初期)→90kW(2.0)→350kW(現在)→900kW(ChaoJi/3.0)
- **特徴**: バッテリー負荷軽減設計、V2H/V2G双方向給電対応
- **次世代**: ChaoJi規格を中国と共同開発、2032-35年に完全移行予定
- **URL**: https://www.chademo.com/

**パナソニック エナジー**
- **供給先**: テスラ(円筒形2170/4680セル)、トヨタ(角型セル)
- **生産拠点**: 米国ネバダ州ギガファクトリー、日本国内工場
- **技術**: ニッケル含有量増加でエネルギー密度向上、熱安定性改善
- **コスト目標**: 2030年までにkWhあたりコスト半減
- **次世代**: 全固体電池の量産技術開発(トヨタと共同)

**トヨタ自動車 バッテリー戦略**
- **PHEV**: プリウスPHEV 13.6kWh(EV走行87km)、アルファードPHEV 18.1kWh(EV走行73km)
- **EV**: bZ4X 71.4kWh(航続距離559km)
- **急速充電**: CHAdeMO対応、30分で80%充電
- **次世代**: 2027-28年に全固体電池搭載EVを実用化予定

### グローバルスタンダード

**Tesla Supercharger V4 + NACS規格**
- **出力**: 最高250kW、将来的に350kW対応予定
- **設置数**: 全世界3万基超、北米・欧州・中国・日本で展開
- **規格**: NACS(North American Charging Standard)を2025年からオープン化
- **採用企業**: GM、フォード、日産、ホンダが2025年から北米向け新型EVでNACS採用
- **特徴**: プラグアンドチャージ(自動認証)、充電口統一(普通+急速)
- **URL**: https://www.tesla.com/supercharger

**CCS Combo(欧州・北米)**
- **規格**: CCS1(北米)、CCS2(欧州)、普通充電と急速充電を1つの充電口で対応
- **出力**: 最大350kW(Ionity、Fastned等)
- **通信**: PLC(Power Line Communication)で車両・充電器間通信
- **設置数**: 欧州Ionityネットワーク1,500箇所以上
- **特徴**: 支払い方法選択可(ステーション直接払い/車両自動課金)

**CATL(中国) + GB/T規格**
- **メーカー**: 世界最大バッテリーメーカー(世界シェア37%)
- **技術**: LFP(リン酸鉄リチウム)電池で低コスト・長寿命を実現
- **供給先**: BYD、テスラ(中国生産Model 3/Y)、ホンダ、日産
- **GB/T規格**: 中国国家標準、普通充電27.7kW、急速充電250kW以上
- **次世代**: ChaoJi規格に統合予定

**LG化学(韓国)**
- **世界シェア**: 14%(2位)
- **技術**: ポーチ型セル、高エネルギー密度
- **供給先**: GM、フォード、ボルボ、ヒョンデ等
- **生産拠点**: GMとの合弁で北米に複数工場展開

**ChaoJi次世代規格**
- **出力**: 50kW~最大900kW(600A×1500V)
- **開発**: 日本CHAdeMO協議会+中国電力企業連合会
- **ロードマップ**: 2020年規格確定→2021年試作→2022年対応車投入→2025年コネクタ縮小→2032-35年完全移行
- **特徴**: コネクタ小型化、ケーブル軽量化、V2X双方向給電対応
- **参加**: ダイムラー、アウディ、コンチネンタル等欧州勢も実証実験参加

## 日本の立ち位置分析: 4つの要点

### 1. CHAdeMO規格での先行優位と課題

**強み**:
- 世界初の急速充電規格、2010年策定・2014年国際標準承認
- 国内12,806基、世界90カ国以上で展開実績
- バッテリー保護を重視した設計思想で信頼性高い
- V2H/V2G双方向給電に早期対応(災害時の電源確保)

**弱み**:
- 欧州ではCCS Comboが主流、北米ではTesla NACSへの移行加速
- 日産が北米2025年新型EVでNACS採用を決定、国内メーカーの戦略転換
- 複数規格乱立で海外展開車両は地域別充電口変更が必要、開発コスト増

**課題**:
- ChaoJi(900kW)実用化が2032-35年と遅く、欧州350kW充電器普及に後れ
- 中国市場ではGB/T規格が主流、ChaoJi統合が進まない可能性
- NACS勢力拡大により、アジア以外でのシェア低下リスク

### 2. バッテリー生産での劣勢

**世界シェア**:
- CATL(中国): 37%
- LG化学(韓国): 14%
- BYD(中国): 12%
- パナソニック(日本): 6%

**国内課題**:
- EV普及が欧中に比べて遅く、国内需要が限定的で量産効果が発揮できず
- 車載バッテリー工場の新設投資で中韓勢に大きく後れ
- LFP電池(低コスト・長寿命)で中国勢が圧倒的優位

**戦略転換**:
- トヨタがパナソニック・プライムプラネットと合弁強化
- 日産がエンビジョンAESC、中国CATL採用拡大
- ホンダがGM-LG合弁電池を北米で調達

### 3. 急速充電インフラ整備の遅れ

**現状**:
- 国内CHAdeMO充電器の多くは50kW出力(初期世代)
- 90kW以上の高出力充電器は限定的(高速道路SA/PA中心)
- 欧州では350kW充電器が既に普及開始

**2025年秋の転機**:
- e-Mobility Power + 東光高岳の350kW充電器展開開始
- 10分充電で400km走行を実現し、欧州水準に追いつく
- 既存50kW充電器の更新需要も発生

**インフラ投資**:
- 政府目標: 2030年までに急速充電器3万基(現在約1.3万基)
- 民間投資: コンビニ・商業施設への設置拡大
- 充電ステーション事業の収益化が課題(従量課金vs定額制)

### 4. 全固体電池への期待

**日本の技術優位**:
- トヨタが2027-28年実用化目標、世界最先端
- 固体電解質で発火リスク低減、-30℃~100℃広範囲動作
- 充電時間10分以下、航続距離1000km超を目指す

**商用化の課題**:
- 量産技術確立が必須、コスト削減が最大の壁
- 2030年代前半まで液系リチウムイオン電池が主流継続の見通し
- 先行投資が実を結ぶかは不透明

## メカトロニクス・品質保証エンジニアの視点

### 品質保証の重要ポイント

1. **バッテリーセル品質管理**
   - 電圧・容量の個体差管理(±2%以内)
   - 内部抵抗のばらつき抑制
   - 製造ロット毎の特性評価・トレーサビリティ確保

2. **温度管理システムの信頼性**
   - 液冷システムの漏れ検査、圧力試験
   - 温度センサー精度校正(±1℃以内)
   - 寒冷地-30℃、酷暑地+50℃での性能維持試験

3. **充電制御の安全性**
   - 過充電・過放電防止のフェイルセーフ設計
   - 充電器・車両間通信の異常検知
   - 絶縁抵抗試験、高電圧安全規格(IEC 62133)準拠

4. **バッテリー劣化予測**
   - 充放電サイクル試験(1000サイクル以上)
   - 容量維持率評価(10年後80%維持目標)
   - カレンダー劣化(保管劣化)の加速試験

1. **劣化予測モデル**

```python
# 充放電サイクル、温度履歴から容量劣化を予測
import pandas as pd
from sklearn.ensemble import RandomForestRegressor

# バッテリー使用履歴データ読込
df = pd.read_csv('battery_log.csv')
# 特徴量: サイクル数、平均温度、充電深度、充電速度
X = df[['cycle', 'avg_temp', 'DOD', 'charge_rate']]
y = df['capacity_retention']  # 容量維持率

# ランダムフォレスト回帰で劣化予測
model = RandomForestRegressor(n_estimators=100)
model.fit(X, y)
```

```python
# 温度・電圧・電流の時系列データから最適充電カーブ算出
import numpy as np
from scipy.optimize import minimize

def battery_degradation(charge_current, temperature):
    # バッテリー劣化モデル(簡易版)

# 制約条件: 充電時間30分以内、温度45℃以下
# 目的関数: 劣化最小化しつつ充電量最大化
```
