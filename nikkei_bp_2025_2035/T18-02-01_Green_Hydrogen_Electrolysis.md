## T18-02-01 グリーン水素製造技術（再エネ電解）

### 💡 全体要約の特徴
太陽光・風力などの**再生可能エネルギー**電力を用いて水を電気分解し、**製造過程でCO2を排出しない**水素（グリーン水素）を製造する技術。主な方式として、**プロトン交換膜（PEM）電解装置**と**アルカリ水電解装置**が実用化段階にある。最大の課題は製造コストの削減であり、2030年までに3ドル/kg（現状5～10ドル/kg）を目指している。IEAは2050年までに世界の水素供給の**70%**をグリーン水素が占めると予測している。

#### 概念図: グリーン水素製造のプロセス
```mermaid
graph TD
    A[太陽光/風力発電] --> B{電力 (再生可能エネルギー)}
    B --> C[水電解装置 (PEM/アルカリ)]
    C --> D[水素 (H2)]
    C -- 廃棄物なし --> E[酸素 (O2)]
    D --> F[貯蔵・輸送/利用 (燃料電池, 産業)]
```

### ⚖️ 日本の立ち位置分析（強み・弱み）
| 項目 | 強み (Strength) | 弱み (Weakness) |
| :--- | :--- | :--- |
| **技術** | PEM電解膜技術、高性能触媒開発における高い技術力。 | 再生可能エネルギーの導入コストが高く、**安価な電力調達が課題**。 |
| **インフラ** | FC（燃料電池）技術との連携による市場創出。 | 大規模な水電解装置の**システムインテグレーション経験**の不足。 |

### 📈 ハイプ・サイクル評価と5つの要点Summary
| 評価項目 | 予測/状況 |
| :--- | :--- |
| **現状** | 啓発期から過度な期待のピーク期へ移行中 | 
| **到達度** | 中 | 
| **普及率** | 伸び始め | 

**5つの要点Summary:**
1. **CO2フリー**: 再生可能エネルギーで水を電気分解し、**CO2を排出しない**唯一の製造技術。
2. **主要技術**: **PEM電解**（応答性・効率性）と**アルカリ電解**（コスト・大規模化）が実用化。
3. **コスト課題**: 2030年までに製造コストを**3ドル/kg**まで引き下げることが最大の課題。
4. **将来性**: 2050年には世界の水素供給の**70%**を占めると予測される脱炭素社会の基盤技術。
5. **鍵**: 大規模な**再エネ発電所**と隣接させた**ギガワット級**の電解プラント建設が鍵。
