# T1-06-04 量子暗号通信・耐量子暗号

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 黎明期～成長期初期（2026-2030年）
**技術成熟度**: TRL 4-7（実証実験～初期展開段階）

## Summary（5つの要点）

1. **量子鍵配送（QKD）**: 量子力学の原理（不確定性原理、量子もつれ）を利用し、盗聴が物理的に不可能な暗号鍵配送を実現。光子1個レベルの微弱光を送信し、盗聴者が観測すると量子状態が変化するため、即座に検知可能。

2. **絶対安全性の保証**: 数学的困難性（素因数分解、離散対数問題）に依存する現行暗号（RSA、ECC）と異なり、量子暗号は物理法則に基づくため、計算能力の進化に左右されない。量子コンピュータの脅威にも耐性。

3. **耐量子暗号（PQC: Post-Quantum Cryptography）**: 古典コンピュータ上で動作し、量子コンピュータ攻撃に耐える暗号アルゴリズム。NIST（米国国立標準技術研究所）が2024年8月に標準化完了（CRYSTALS-Kyber、CRYSTALS-Dilithium、SPHINCS+）。

4. **実用化の課題**: QKDは専用光ファイバー敷設が必要で、長距離通信（100km以上）が困難。量子中継器の開発により1,000km超の通信を目指す。衛星QKD（日本のQUESTプロジェクト、中国の墨子号）により地球規模の量子暗号網を構築中。

5. **金融・インフラへの適用**: 中央銀行間決済、株式取引、機密情報通信、重要インフラ制御（電力、通信、交通）でのセキュリティ確保に不可欠。"Harvest Now, Decrypt Later"攻撃（現在暗号化データを保存し、将来の量子コンピュータで解読）への対策として緊急性が高い。

## 具体的プロダクト事例

### 日本企業の先進事例

#### 1. **NTT「量子暗号通信システム（QKD）」**
- **URL**: https://www.rd.ntt/research/JN202207_2212.html
- **概要**: NTTの光通信技術と量子技術を融合した世界最長距離QKDシステム。
- **技術特徴**: 
  - 800km超の長距離QKD実証（世界記録、2022年）
  - 既設光ファイバーでの動作（専用線不要）
  - 量子中継器の開発（1,000km超を目指す）
- **導入実績**: 
  - 東京～大阪間QKDネットワーク実証実験
  - 金融機関との共同実証（みずほ銀行、三菱UFJ銀行）

#### 2. **東芝「Quantum Key Distribution System」**
- **URL**: https://www.global.toshiba/ww/products-solutions/security-ict/quantum-cryptography.html
- **概要**: 東芝が世界で初めて商用化したQKDシステム（2023年発売）。
- **技術特徴**: 
  - Twin-Field QKD方式（長距離・高速通信）
  - 鍵生成速度：10Mbps以上
  - コンパクト設計（19インチラック搭載可能）
- **導入実績**: 
  - 英国BT（British Telecom）との共同実証
  - 欧州金融機関での商用導入
  - 日本国内では電力会社、通信キャリアでの実証

#### 3. **三菱電機「衛星量子暗号通信システム（QUEST）」**
- **URL**: https://www.mitsubishielectric.co.jp/news/2020/0710-b.html
- **概要**: JAXA、NICTと共同開発した小型衛星搭載QKDシステム。
- **技術特徴**: 
  - 衛星-地上局間QKD（距離1,000km超）
  - 小型軽量化（50kg以下）
  - 大気揺らぎ補正技術
- **目標**: 2025年打ち上げ予定の技術実証衛星で実証

#### 4. **NEC「耐量子暗号ライブラリ（liboqs-NEC）」**
- **URL**: https://jpn.nec.com/press/202408/20240823_01.html
- **概要**: NIST標準化耐量子暗号アルゴリズムの日本版実装。
- **技術特徴**: 
  - CRYSTALS-Kyber、CRYSTALS-Dilithium、SPHINCS+実装
  - 組み込みシステム向け最適化（IoT、制御システム対応）
  - オープンソース公開
- **適用分野**: 5G/6G通信、IoTデバイス、車載システム

### グローバルスタンダード

- **URL**: https://openqkd.eu/

  - 5G/6Gネットワークとの統合
- **目標**: 2027年までに欧州全域カバーの量子インターネット構築

#### 3. **米国「NIST 耐量子暗号標準（FIPS 203, 204, 205）」**
- **URL**: https://csrc.nist.gov/projects/post-quantum-cryptography

- **技術特徴**: 
  - FIPS 203（CRYSTALS-Kyber）：公開鍵暗号
  - FIPS 204（CRYSTALS-Dilithium）：デジタル署名
  - FIPS 205（SPHINCS+）：ハッシュベース署名
- **影響**: 全世界の暗号システムが2030年までに移行必須

#### 4. **ID Quantique（スイス）「Cerberis XG QKD」**
- **URL**: https://www.idquantique.com/quantum-safe-security/products/cerberis-xg-qkd-platform/

  - Plug-and-Play型QKDシステム
  - 既存ネットワークへの統合が容易
  - 金融、政府、通信キャリア向け

## 全体要約（5つの要点）

### 1. 技術の本質
量子暗号通信は「盗聴不可能」という絶対安全性を物理法則で保証する革命的技術。量子コンピュータが現行暗号（RSA、ECC）を破る「Yデー（Y2Q）」に備え、政府・金融機関・重要インフラは2030年までに耐量子暗号へ移行必須。"Harvest Now, Decrypt Later"攻撃への対策が喫緊の課題。

### 2. コア技術要素
- **量子鍵配送（QKD）**: BB84プロトコル、E91プロトコル、Twin-Field QKD
- **単一光子源**: 量子ドット、ダウンコンバージョン、弱コヒーレント光
- **量子中継器**: 量子メモリ、量子もつれ交換、誤り訂正
- **衛星QKD**: 自由空間光通信、適応光学系、大気揺らぎ補正
- **耐量子暗号（PQC）**: 格子暗号、符号暗号、多変数多項式暗号、ハッシュベース署名

### 3. 応用分野
- **金融**: 中央銀行間決済、株式取引、国際送金
- **政府**: 外交機密通信、国家安全保障情報
- **重要インフラ**: 電力網制御、通信ネットワーク管理、交通制御
- **医療**: 患者個人情報、ゲノムデータ保護
- **企業**: 営業秘密、知的財産、M&A情報

### 4. 主要プレイヤー
- **日本**: NTT、東芝、三菱電機、NEC、NICT（情報通信研究機構）
- **中国**: 中国科学技術大学、QuantumCTek、国盾量子
- **欧州**: ID Quantique（スイス）、Thales（フランス）、Fraunhofer（ドイツ）
- **米国**: IBM、Google（量子コンピュータ開発）、NIST（標準化）

### 5. 将来展望
2026-2028年、金融機関・政府での商用QKD導入開始。2030年までに耐量子暗号（PQC）が全世界のITシステムに実装完了。衛星QKDにより地球規模の量子暗号網が構築され、国家間・大陸間の機密通信が可能に。2035年以降、量子中継器の実用化により1,000km超の長距離QKDが実現し、量子インターネットの基盤が完成。

## 日本の立ち位置（4つの要点）

### 1. 強み：光通信技術とQKD研究

NTTは光通信技術で世界トップクラスであり、既設光ファイバーで動作するQKDシステムを開発。800km超の長距離QKD実証で世界記録保持。東芝は商用QKDシステムを世界で初めて販売し、欧州市場で実績。NICT（情報通信研究機構）は量子通信研究で30年以上の蓄積。小型衛星技術（JAXA、三菱電機）と組み合わせた衛星QKDで中国に対抗。

### 2. 弱み：PQC標準化で米国に後れ

耐量子暗号（PQC）の国際標準化はNISTが主導し、米国企業・大学が中心。日本からの提案は採用されず、標準化プロセスへの関与が限定的。NECはliboqs-NECを公開したが、実装面での対応にとどまる。格子暗号、符号暗号等の数学的基礎研究で欧米に後れ。暗号理論研究者の絶対数が不足。

### 3. 機会：デジタル庁・政府調達での展開

デジタル庁は「政府情報システムにおける暗号技術の利用方針」でPQC移行を明記。2030年までに中央省庁・自治体の全システムがPQC対応必須。金融庁もFISC安全対策基準にPQC推奨を追記。製造業、インフラ事業者も経済安全保障の観点から量子暗号導入を検討中。国内市場拡大がベンダー育成の好機。

### 4. 脅威：中国の量子技術覇権

中国は国家戦略として量子技術に巨額投資（年間100億ドル以上）。墨子号衛星QKDで先行し、2030年までに全世界カバーの量子通信網構築を目指す。中国企業（QuantumCTek、国盾量子）は商用QKD市場シェアを拡大中。米中対立により、日本は中国製量子技術の調達禁止圧力を受ける可能性。一方で、米国NI STが主導するPQC標準に依存するリスクも。技術的自立性確保が急務。
