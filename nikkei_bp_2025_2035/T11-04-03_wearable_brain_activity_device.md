# T11-04-03 脳活動測定ウェアラブルデバイス

## 技術の位置づけ

近赤外分光法（fNIRS）や脳波計（EEG）をウェアラブル化し、日常生活中の脳活動をモニタリングする技術。MRI・PETのような大型装置に比べ、非侵襲・低コスト・可搬性に優れ、認知機能低下の早期兆候検知、リハビリテーション効果測定、日常的脳健康管理に活用される。

## 5つの要点サマリー

1. **非侵襲的リアルタイム測定**: 近赤外光（700-900nm）を用いて脳血流中のヘモグロビン酸素化状態を測定。放射線や薬品を使わず安全で、日常生活に近い環境下で脳活動を可視化

2. **ウェアラブル化の進展**: 従来の据え置き型装置から、軽量・無線通信対応のポータブル型へ進化。日立グループからスピンオフしたNeUのHOT-2000は、簡単装着でリアルタイム計測を実現

3. **多様な計測シーン対応**: 歩行中、運転中、リハビリテーション中など、体を動かしながらの脳活動測定が可能。MRI/PETでは不可能な「動作時の脳機能」を評価できることが最大の強み

4. **社会脳科学への応用**: 母子間コミュニケーション、チーム作業時の脳同期、スポーツ選手の集中状態など、社会的インタラクション中の脳活動研究に不可欠なツールとして注目

5. **認知症早期検知への期待**: 認知機能低下に伴う脳活動パターンの変化を日常的にモニタリング。デジタルバイオマーカーとして、MCIや軽度認知症の早期発見・進行予測に活用される可能性

## 具体的プロダクト事例

### 日本企業の先進事例

1. **NeU ウェアラブル光トポグラフィ HOT-2000**
   - URL: https://neu-brains.co.jp/solution/nirs/
   - 概要: 日立製作所が1995年に世界初開発したfNIRS装置の技術を継承。小型・ウェアラブル化を実現
   - 特徴: 重量約200g、無線通信対応。リアルタイムで脳活動量変化を計測。専用アプリで脈拍・頭部加速度データも同時取得
   - 応用: 医療研究、リハビリテーション、スポーツ医学、心理学、教育学、自動車・産業分野での認知負荷評価

2. **島津製作所 LABNIRS（ラブニルス）**
   - URL: https://www.an.shimadzu.co.jp/products/life-science-lab-instruments/imaging/fnirs/
   - 概要: fMRIやEEGとの同時計測が可能な高性能fNIRS装置。据え置き型だが、拘束が少なく自然に近い環境で計測
   - 特徴: 酸素化ヘモグロビン（Oxy-Hb）、脱酸素化ヘモグロビン（Deoxy-Hb）、総ヘモグロビン（Total-Hb）の変化量を同時表示
   - 応用: 基礎研究、脳機能マッピング、医療研究、リハビリテーション研究

3. **理化学研究所 / 脳の学校 ベクトル法fNIRS**
   - URL: https://nonogakko.com/
   - 概要: 加藤俊徳医師が開発した、酸素交換波動方程式に基づくfNIRS解析法。脳番地（領域）ごとの機能評価が可能
   - 特徴: 脳のMRI画像と統合し、個人の脳の成長・老化を定量評価。「脳の学校」でパーソナル脳トレーニングプログラムを提供

4. **オムロン / 脳活動センサー（開発中）**
   - 概要: ヘルスケア機器メーカーとして、家庭用脳活動モニタリングデバイスの開発を検討。睡眠・ストレス・認知機能の日常的管理を目指す

### グローバルスタンダード

1. **Kernel / Flow 2 (米国)**
   - URL: https://www.kernel.com/
   - 概要: 2016年設立のニューロテックスタートアップ。fNIRSとEEGを統合したウェアラブルデバイス
   - 特徴: 3.7Hzサンプリングレートで頭部全体の血流測定をカバー。携帯性を高めた設計で研究機関・企業に展開
   - 応用: うつ病診断補助、薬剤効果検証、アルコールが脳機能に及ぼす影響研究（2023年Scientific Reports掲載）

2. **Artinis / Brite (オランダ)**
   - URL: https://www.artinis.com/
   - 概要: NIRS研究の第一人者Dr. Willy Colierが設立。脳酸素化測定用のウェアラブル・マルチチャンネルfNIRSデバイス
   - 特徴: 最大30台のfNIRSデバイスをシームレスに同期可能。ハイパースキャニング研究（複数人の脳を同時計測）に最適
   - 応用: 世界40カ国、350以上の大学・クリニック・企業で採用。スポーツ科学、リハビリテーション、BMI研究

3. **NIRx / NIRSport2 (米国・ドイツ)**
   - URL: https://nirx.net/
   - 概要: ポータブルfNIRSシステム。最大40チャンネルの計測に対応し、EEGとの同時計測も可能
   - 特徴: オープンソースソフトウェア（HOMER3、NIRS-SPM）に対応。研究用途に広く採用
   - 応用: 神経科学研究、ブレイン・コンピュータ・インターフェース（BCI）、認知心理学

4. **Obelab / NIRSIT (韓国)**
   - URL: https://www.obelab.com/
   - 概要: 超軽量（300g以下）のウェアラブルfNIRSデバイス。スマートフォンアプリで簡単操作
   - 特徴: 低価格帯で教育・研究機関に普及。新興国でのfNIRS研究を加速
   - 応用: 赤ちゃん・幼児の脳発達研究、母子関係、社会的インタラクション研究

## My Notes

### 品質保証エンジニアの視点

**測定精度とノイズ対策**
- モーションアーチファクト（体動によるノイズ）の除去アルゴリズム
- 皮膚血流、頭蓋骨血流の影響を分離する信号処理技術
- 周囲光（太陽光、室内照明）の影響評価と補正
- Pythonによる信号処理: バンドパスフィルタ、ICA（独立成分分析）、ウェーブレット変換

**デバイスの信頼性評価**
- 送受光器の位置再現性（装着のたびに位置がずれないか）
- 無線通信の安定性（データ欠損、遅延の発生頻度）
- バッテリー持続時間と計測時間の制約
- 長期使用における光源（LED）の出力劣化

**標準脳座標系への対応**
- 国際10-20法（脳波の電極配置基準）に準拠した送受光器配置
- デジタイザーによる3次元位置測定と標準脳座標系（MNI、Talairach）へのレジストレーション
- 確率的レジストレーション法（POTATo、HOMER、NIRS-SPM）の精度検証

**データ解析とPython応用**
- 時系列データの前処理（トレンド除去、スパイクノイズ除去）
- 脳活動パターンの統計解析（t検定、ANOVA、多重比較補正）
- 機能的結合解析（DMN、DAN、CENなどの脳ネットワーク）

### 化学系大学生への示唆

**光化学・分光学の応用**
- 近赤外光とヘモグロビンの相互作用（吸収スペクトルの変化）
- Beer-Lambert則に基づく定量測定の原理
- 生体組織における光散乱・吸収の物理モデル

**生理学・神経科学**
- 神経活動と脳血流の相関（神経血管カップリング）
- 酸素代謝とヘモグロビン動態
- 脳血流自動調節機構と認知機能の関係

**信号処理とデータサイエンス**
- 時系列データのフーリエ変換、ウェーブレット変換
- ノイズ除去のためのフィルタ設計
- 機械学習による脳状態分類（リーマン幾何学的アプローチ）

### スイミング中学生への応用

**スポーツ科学への応用**
- トレーニング中の脳活動モニタリング（集中力、疲労度の可視化）
- 競技パフォーマンスと脳血流の関係分析
- 脳振とう後の脳機能評価と復帰判定

**日常的な健康管理**
- ストレス・睡眠の質と脳活動の関係
- 勉強中の集中力モニタリングと効率的学習法の発見
- 将来のニューロフィードバックトレーニングへの応用

## 日本の立ち位置と強み・弱み（4つの要点）

### 強み

1. **fNIRS技術の先駆性**: 1995年に日立製作所が世界初のfNIRS装置を開発・実用化。「光トポグラフィ」として商標登録し、技術の礎を築いた。NeUが技術を継承し、ウェアラブル化を推進

2. **精密計測技術とハードウェア品質**: 島津製作所、日立グループの光学測定技術、センサー技術は世界トップレベル。デバイスの測定精度・再現性・長期安定性に優れる

3. **医療・介護分野での実証研究**: 高齢化社会の日本では、認知症・リハビリテーション研究でfNIRS活用実績が豊富。臨床現場でのニーズを反映した製品開発が可能

4. **基礎研究と標準化への貢献**: 理化学研究所、東京大学、慶應義塾大学等の研究機関がfNIRS解析手法（POTAToなど）を開発し、国際標準化に貢献

### 弱み

1. **ウェアラブル化・コンシューマー展開の遅れ**: 欧米企業（Kernel、Artinis）は携帯性・UI/UXを重視した製品開発を加速。日本は研究機関向けB2B中心で、一般消費者向けデバイスが少ない

2. **ソフトウェア・データ解析基盤の弱さ**: オープンソースソフトウェア（HOMER、NIRS-SPM）は欧米が主導。日本はハードウェアに強いが、AIアルゴリズム、クラウド解析基盤で後れ

3. **ビジネスモデルとスケーラビリティ**: 米国スタートアップは研究機関だけでなく、スポーツチーム、企業ウェルネスプログラム、メンタルヘルス市場を開拓。日本は市場展開が限定的

4. **規制とデータ利活用**: 医療機器承認の壁、個人健康データの利活用に関する法整備が遅れ、イノベーションスピードが遅い。欧米は研究用から診断補助への移行が速い

## 品質保証エンジニアとしての技術調査ポイント

### 即座に調査すべき項目

1. **信号品質の評価指標**
   - SNR（Signal-to-Noise Ratio）の定量評価
   - モーションアーチファクト混入率と除去効率
   - 測定部位（前頭葉、側頭葉等）による信号品質の差異

2. **デバイスのキャリブレーション**
   - ファントム（脳組織を模した光学ファントム）による精度検証
   - 送受光器間距離と光路長の関係式検証
   - LED光源の出力安定性と経時変化

3. **標準化とトレーサビリティ**
   - 測定値の国際標準（ISO規格等）への対応
   - 施設間・装置間での測定値の互換性検証
   - データフォーマットの標準化（BIDS-fNIRS）

### Pythonライブラリの活用

- **MNE-NIRS**: fNIRSデータの前処理、解析、可視化専用ライブラリ
- **scipy.signal**: バンドパスフィルタ、ウェーブレット変換、ICA
- **nilearn**: 脳画像データの統計解析、機能的結合解析
- **pandas, numpy**: 時系列データの処理、統計解析
- **matplotlib, seaborn**: 脳活動パターンの可視化
