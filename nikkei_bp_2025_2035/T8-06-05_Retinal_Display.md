## ハイプ・サイクル位置づけ

**黎明期 → 過度な期待のピーク期** (2025年時点)

- QD Laser RETISSA Display IIが医療・産業向けに実用化段階
- 視力補正不要という革新性に注目、ただし価格・視野角が課題
- 2026-2030年にARグラス統合により本格普及見込み

---

## Summary: 5つの要点

1. **視力に依存しない映像投影**: レーザー光を直接網膜に走査投影、近視・遠視・乱視・老眼の影響を受けずクリアに視認。眼鏡・コンタクト不要
2. **ピント位置不問の視界重畳**: 投影映像は常に焦点が合った状態で視界に重なるため、現実世界と仮想情報を同時に鮮明視認
3. **ロービジョン支援への応用**: 角膜・水晶体疾患(白内障、円錐角膜等)で視力低下した方への視覚支援機器として実用化。健康保険適用検討中
4. **小型軽量化の進展**: MEMS+量子ドットレーザーで超小型プロジェクタ実現。眼鏡型フレームに内蔵可能(65×20×83.5mm)
5. **次世代ARグラスの鍵技術**: MicroLED網膜投影と組み合わせた究極の没入型ARグラスとして期待。視力補正機能統合でメガネ不要社会へ

## 技術詳細分析

### **網膜投影の原理**

**光学系構成**:
1. RGB量子ドットレーザー(波長: R 638nm, G 520nm, B 445nm)
2. MEMSミラー(2軸、共振周波数: 水平30kHz、垂直60Hz)
3. 光学反射ミラー(瞳孔へ導光)
4. 瞳孔通過後、網膜上で結像

**走査方式**:
- ラスタースキャン(CRT方式と同様)
- 水平1280画素×垂直720画素を1/60秒で描画
- 各画素を順次レーザー照射して描画

**視力に依存しない理由**:
- 水晶体(レンズ)の屈折を経由せず網膜に直接到達
- 角膜の歪み(乱視)も影響しない
- 老眼(毛様体筋衰え)も無関係

### **安全性**

**レーザークラス1**:
- IEC60825-1規格準拠
- 蛍光灯と同等以下の微弱光
- 連続視聴でも眼へのダメージなし

**長期使用の安全性**:
- 10年以上の研究開発で有害事象報告なし
- 医療機器認証取得に向け治験実施中

### **現行製品の制約**

**視野角**: 26度(対角)
- 通常視野(約200度)の13%
- スマートフォン画面を50cm先で見た程度
- AR情報表示には十分だが、没入型VRには不足

**解像度**: 720p相当
- 文字の可読性は確保
- 動画視聴は実用レベル
- 4K化は次世代課題

**重量**: 投影ユニット約100g
- 長時間装着で負担
- MicroLEDレーザー化で50g以下目標

## 品質保証エンジニアの視点

### **品質評価項目**

1. **光学性能**
   - レーザー出力安定性: ±5%以内
   - 色度座標精度: Δu'v'<0.005
   - 輝度均一性: 画面全域で±10%以内
   - コントラスト比: >100:1

2. **走査精度**
   - MEMSミラー位置精度: ±0.1度
   - フレームレート: 60fps±1%
   - ジッター(揺らぎ): <1pixel

3. **安全性**
   - レーザークラス1適合性試験
   - 長期曝露試験(1,000時間)
   - 異常動作時の自動停止機能

4. **ユーザビリティ**
   - 装着快適性: NASA-TLX<50
   - 瞳孔位置ずれ許容範囲: ±3mm
   - キャリブレーション不要化

### **Python活用例**

```python
# 網膜投影ディスプレイの画質評価
import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import welch

def evaluate_temporal_stability(luminance_time_series, sampling_rate_hz=60):
    """
    時間的輝度安定性評価(フリッカー検出)
    """
    # パワースペクトル密度計算
    freqs, psd = welch(luminance_time_series, fs=sampling_rate_hz, nperseg=sampling_rate_hz*2)
    
    # フリッカー周波数帯域(8-60Hz)のパワー
    flicker_band = (freqs >= 8) & (freqs <= 60)
    flicker_power = np.trapz(psd[flicker_band], freqs[flicker_band])
    
    # 総パワーに対する比率
    total_power = np.trapz(psd, freqs)
    flicker_ratio = flicker_power / total_power
    
    # フリッカー指標(低いほど良い)
    flicker_index = flicker_ratio * 100
    
    return flicker_index

def measure_retinal_scanning_jitter(beam_position_data, ideal_trajectory):
    """
    レーザービーム走査のジッター(位置ずれ)測定
    """
    # 理想軌跡からの偏差
    deviation = beam_position_data - ideal_trajectory
    
    # RMS誤差(pixel単位)
    rms_jitter = np.sqrt(np.mean(deviation**2))
    
    # 最大偏差
    max_jitter = np.max(np.abs(deviation))
    
    return {'rms_jitter_pixel': rms_jitter, 'max_jitter_pixel': max_jitter}

def assess_accommodation_independence(visual_acuity_data, diopter_range):
    """
    視力(調節力)依存性評価
    網膜投影が本当に視力に依存しないかの検証
    """
    # 各屈折異常(ジオプター)での視力測定値
    # 理想的には全て一定
    
    mean_acuity = np.mean(visual_acuity_data)
    std_acuity = np.std(visual_acuity_data)
    
    # 変動係数(CV)で評価
    cv_percent = (std_acuity / mean_acuity) * 100
    
    # CV<5%なら視力依存性無しと判定
    is_independent = cv_percent < 5
    
    return {'cv_percent': cv_percent, 'is_accommodation_independent': is_independent}
```

## ロービジョン支援への応用

### **適応疾患**

**角膜疾患**:
- 円錐角膜(角膜が円錐状に突出)
- 角膜混濁・瘢痕
- → レーザー光が角膜を通過せず網膜直達で視認可能

**水晶体疾患**:
- 白内障(手術不適応患者)
- 水晶体脱臼
- → 水晶体の屈折を経由しないため影響なし

**屈折異常**:
- 強度近視・遠視(-20D ~ +10D)
- 不正乱視
- → 眼鏡・コンタクト不要で鮮明視認

**不適応疾患**:
- 網膜剥離・網膜変性
- 視神経疾患
- → 網膜自体に問題がある場合は効果なし

### **RD2CAMアクセサリー活用**

**機能**:
- 最大3倍ズーム(遠くの物を見る)
- 1300万画素高画質
- 白黒反転(まぶしさ軽減)
- 明るさ調整(個別最適化)

**用途**:
- 演劇鑑賞、スポーツ観戦
- 美術館での絵画鑑賞
- 旅行先での風景撮影

## 次世代ARグラスへの統合

### **理想的なARグラスの条件**

1. **視野角**: 50度以上(現状26度→目標達成へ)
2. **解像度**: 2K以上(現状720p→MicroLED化で実現)
3. **重量**: 50g以下(現状100g→小型化必須)
4. **バッテリー**: 8時間以上(現状3.3時間→省電力化)
5. **視力補正**: 全自動(キャリブレーションレス)

### **Apple/Meta ARグラスへの可能性**

**Appleの動向**:
- Vision Proは現在、導光板+パンケーキレンズ方式
- 将来世代で網膜投影統合の可能性(特許出願複数)
- QDレーザー技術のライセンス or M&A可能性

**Metaの動向**:
- Orionプロトタイプは導光板方式
- 視力補正機能への強いニーズ(ザッカーバーグも言及)
- 網膜投影はポスト導光板の有力候補

## 医療機器認証への道筋

### **現状**
- RETISSA Display IIは現在「医療機器ではない」と明示
- 視覚支援機器として眼鏡店で販売

### **医療機器化のメリット**
- 健康保険適用(患者負担1-3割)
- 障害者総合支援法の補装具費支給対象
- 医療機関での処方・フォローアップ

### **必要なステップ**
1. 薬機法クラスII医療機器申請
2. 臨床試験(治験)実施
3. PMDA(医薬品医療機器総合機構)審査
4. 製造販売承認取得
5. 保険収載申請

### **想定スケジュール**
- 2026年: 治験開始
- 2027-2028年: PMDA審査
- 2029年: 承認・保険収載
- 2030年~: 全国眼科での処方開始
