# T2-03-03 昆虫タンパク質生産技術(コオロギ、ミールワーム)

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 成長期初期(2026-2030年)
**技術成熟度**: TRL 6-7(実証実験から初期商業展開段階)

## Summary(5つの要点)

1. **超高効率なタンパク質生産**: コオロギ、ミールワームは飼料転換効率が牛の12倍、豚の4倍、鶏の2倍と極めて高く、少ない飼料で大量のタンパク質を生産可能。CO2排出量は牛肉の1/10以下で、環境負荷が極めて低い

2. **自動飼育システムによる大量生産**: 温度、湿度、餌供給、収穫を完全自動化した垂直農法型飼育システムにより、小面積で大量生産を実現。都市近郊での生産が可能で、輸送コストも削減

3. **高栄養価と機能性**: タンパク質含有率60-70%(乾燥重量)で牛肉(約20%)の3倍以上。必須アミノ酸、オメガ3脂肪酸、ビタミンB12、鉄分、カルシウムが豊富。キチンは食物繊維として腸内環境改善効果

4. **粉末加工技術による食品応用**: 乾燥・粉末化技術により、プロテインバー、パスタ、パン、クッキー、スナック菓子など多様な食品に配合可能。無味無臭化技術により、消費者抵抗感を低減

5. **欧州・アジアでの市場拡大**: EU、タイ、韓国、シンガポールで食用昆虫の規制整備が進み、市場が急拡大。世界市場規模は2025年15億ドル、2030年80億ドルに達する見込み

## 具体的プロダクト事例

### 日本企業の先進事例

#### 1. **グリラス コオロギ食品シリーズ**
- **URL**: https://gryllus.jp/
- **概要**: 徳島大学発スタートアップ。フタホシコオロギの自動飼育システムを開発し、食品原料として供給
- **技術特徴**:
  - IoT制御による完全自動飼育システム(温度28度、湿度70%に自動制御)
  - フタホシコオロギの大量生産技術(年間数トン規模)
  - コオロギパウダー、コオロギプロテインバー、コオロギせんべいを製造
  - 無印良品「コオロギせんべい」で話題に
- **成果**: 2024年までに累計調達額10億円超。全国の小売店で販売拡大

#### 2. **BugMo(バグモ) ミールワーム生産**
- **URL**: https://bugmo.jp/
- **概要**: ミールワーム(ゴミムシダマシの幼虫)の大規模生産システムを開発
- **技術特徴**:
  - 垂直農法型飼育施設で省スペース大量生産
  - ミールワームパウダーの製造・販売
  - 家畜飼料、養殖魚飼料としても供給
  - 廃棄パンを飼料として活用する循環型モデル
- **成果**: 年間100トン生産能力を構築。2025年から欧州輸出開始

#### 3. **TAKEO コオロギラーメン、コオロギカレー**
- **URL**: https://takeo.tokyo/
- **概要**: 食用昆虫の専門企業。コオロギを使った調理済み食品を開発・販売
- **技術特徴**:
  - コオロギ粉末を麺、ルウに配合したラーメン、カレー製品
  - タイ産コオロギを輸入し、国内で加工
  - オンラインストア、イベント出店で販売
  - 昆虫食の啓発活動を積極展開
- **成果**: 累計販売数10万食突破。メディア露出多数

#### 4. **エコロギー コオロギ自動飼育ロボット**
- **URL**: https://ecologgie.com/
- **概要**: コオロギ飼育プロセスを完全ロボット化するシステムを開発
- **技術特徴**:
  - 餌供給、温湿度管理、収穫を完全自動化
  - AIカメラによるコオロギ成長モニタリング
  - 最適な収穫タイミングをAIが判定
  - 小規模農家向けに低コストロボットを提供
- **成果**: 2024年から農家向けリース開始

#### 5. **日本農産工業 昆虫飼料プロジェクト**
- **URL**: https://www.nosan.co.jp/
- **概要**: 飼料メーカーがミールワームを家畜・養殖魚飼料として生産
- **技術特徴**:
  - ミールワームの大量生産技術(年間1,000トン規模)
  - 鶏、豚、養殖魚向け飼料として供給
  - 魚粉代替としてコスト削減に貢献
  - 食品廃棄物を飼料として活用する循環モデル
- **成果**: 国内養鶏場、養殖場での採用が拡大中

### グローバルスタンダード

#### 1. **Ynsect ミールワーム生産(フランス)**
- **URL**: https://www.ynsect.com/
- **概要**: 世界最大の昆虫生産企業。垂直農法で年間20万トンのミールワームを生産
- **技術特徴**:
  - 高さ40mの垂直農法施設で大量生産
  - 完全自動化された飼育・収穫システム
  - ペットフード、家畜飼料、人間用食品原料として供給
  - 昆虫由来の肥料も製造(フラス・糞を有機肥料に加工)
- **導入実績**: 2021年シリーズC累計調達額5億ドル超。欧州全域に供給

#### 2. **Protix ブラックソルジャーフライ生産(オランダ)**
- **URL**: https://protix.eu/
- **概要**: ブラックソルジャーフライ(アメリカミズアブ)の幼虫を飼料用タンパク質源として生産
- **技術特徴**:
  - 食品廃棄物を飼料として活用する循環型モデル
  - 年間1万トン生産能力(欧州最大規模)
  - 養殖魚、ペットフード、家畜飼料に供給
  - 昆虫由来の脂質(ローリン酸豊富)も商品化
- **導入実績**: Bühler(スイス)と提携し、世界展開中

#### 3. **Chapul コオロギプロテインバー(米国)**
- **URL**: https://www.chapul.com/
- **概要**: コオロギ粉末を使ったプロテインバーのパイオニア企業
- **技術特徴**:
  - コオロギ粉末、ナッツ、ドライフルーツを配合したプロテインバー
  - タンパク質10g/バー、必須アミノ酸豊富
  - フレーバーは5種類(チョコレート、ピーナッツバター等)
  - 「Shark Tank」(米国TV番組)出演で話題に
- **導入実績**: 全米のスーパーマーケット、オンラインストアで販売

#### 4. **Cricket One コオロギ粉末生産(タイ)**
- **URL**: https://cricket-one.com/
- **概要**: タイ最大のコオロギ生産企業。年間1,000トン規模で生産
- **技術特徴**:
  - タイの伝統的な昆虫食文化を活かした生産
  - コオロギ粉末を欧州、日本、米国に輸出
  - オーガニック認証取得済み
  - B2B向けに食品メーカーに原料供給
- **導入実績**: 欧州、アジアの食品メーカー100社以上に供給

#### 5. **Entomo Farms コオロギ・ミールワーム生産(カナダ)**
- **URL**: https://entomofarms.com/
- **概要**: 北米最大の昆虫生産企業。コオロギ、ミールワームを生産
- **技術特徴**:
  - 6万平方フィート(約5,500m²)の生産施設
  - 有機飼料を使用した高品質昆虫生産
  - 粉末、焙煎、冷凍など多様な形態で供給
  - 人間用食品、ペットフード、家畜飼料に対応
- **導入実績**: 北米の食品メーカー、ペットフードメーカーに供給

## 全体要約の特徴

1. **極めて高い環境効率性**: CO2排出量、水使用量、土地使用量が畜産の1/10以下で、地球環境への負荷が極めて低い。飼料転換効率が牛の12倍、豚の4倍と圧倒的に高く、資源効率が優れる

2. **自動化技術による生産性向上**: 温度、湿度、餌供給、収穫を完全自動化したシステムにより、人手をかけずに大量生産が可能。垂直農法により小面積で高密度生産を実現

3. **高栄養価と健康メリット**: タンパク質含有率60-70%と極めて高く、必須アミノ酸、オメガ3脂肪酸、ビタミンB12、鉄分、カルシウムが豊富。キチンは食物繊維として腸内環境改善効果

4. **多様な食品応用**: 粉末化技術により、プロテインバー、パスタ、パン、クッキー、スナック菓子など多様な食品に配合可能。無味無臭化により、消費者抵抗感を低減

5. **循環型モデルの構築**: 食品廃棄物を昆虫の飼料として活用し、昆虫の糞(フラス)を有機肥料として利用する循環型モデルを構築。廃棄物ゼロ社会への貢献

## 日本の立ち位置、弱み強みのSummary

### 1. 強み: 自動化技術と食品加工技術の優位性

日本は産業用ロボット、IoT制御システムで世界トップレベルの技術を持ち、昆虫飼育の完全自動化システム開発で優位性がある。グリラス、エコロギーなどが開発した自動飼育ロボットは、温湿度管理、餌供給、収穫を完全自動化し、生産性を大幅に向上させている。また、食品加工技術(粉末化、無味無臭化、配合技術)でも日本企業は高い技術力を持ち、消費者受容性の高い製品開発が可能。

### 2. 弱み: 生産規模と消費者受容性の課題

日本の昆虫生産企業は年間数トン~数十トン規模にとどまり、欧州のYnsect(年間20万トン)、Protix(年間1万トン)と比較して生産規模が小さい。資金調達額も欧米企業の1/10以下で、大型設備投資が困難。また、日本では昆虫食への消費者抵抗感が強く、市場形成が遅れている。無印良品「コオロギせんべい」が話題になったが、一部消費者から批判的な意見もあり、普及には時間がかかる見込み。

### 3. 機会: アジア市場での伝統的昆虫食文化の活用

タイ、ラオス、カンボジア、ベトナムなど東南アジアでは伝統的に昆虫食文化があり、市場受容性が高い。日本企業が東南アジアに生産拠点を設立し、現地市場および欧州・日本への輸出を行うビジネスモデルが有効。タイのCricket Oneは日本、欧州に輸出し、成功している。また、日本国内でも高タンパク食品へのニーズが高まっており、スポーツ栄養、高齢者栄養市場での展開が期待される。

### 4. 脅威: 欧州企業の大規模投資と規制整備の遅れ

欧州企業は数億ドル規模の資金調達により、大型垂直農法施設を建設し、量産体制を構築している。YnsectはIPOを目指しており、上場により更なる資金調達が可能となる。日本企業は資金調達力で劣り、生産規模、コスト競争力で大きく後れを取るリスク。また、日本では食用昆虫の安全性評価、食品表示基準が未整備で、商業化に時間がかかる。EUは2021年に乾燥ミールワームを新規食品(Novel Food)として承認しており、規制整備で先行している。
