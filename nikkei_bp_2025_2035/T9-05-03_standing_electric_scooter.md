# T9-05-03 立ち乗り電動スクーター・セグウェイ型モビリティ

## Summary - 技術概要5つの要点

1. **重心移動で操作する自立型移動体** - 立ったまま乗車、重心移動で前後移動・カーブ・スピン、時速10-20km、航続距離20-30km、ジャイロセンサーで自動バランス制御
2. **工場・施設内移動に特化した業務用途** - Segway、Ninebot、Whillが製品化、工場内移動、大型施設巡回、警備、観光案内、空港・テーマパークで活用
3. **バリアフリー対応の電動車いす進化** - Whill Model Cが介護・福祉分野で普及、歩道走行可能(時速6km)、シニア・障がい者の移動手段として規制整備進行
4. **公道走行規制が普及の障壁** - 日本では原則私有地・認定エリアのみ、欧米も都市部で規制強化、歩行者安全確保とモビリティ普及のバランスが課題
5. **市場は自立バランス型から多様化** - 電動スクーター市場2024年約3,340億円、2032年7,440億円予測(CAGR10.5%)、立ち乗り型は一部セグメント、座席付き・折りたたみ式が主流

## 製品・プロダクト事例

### 日本企業の先進事例

#### WHILL株式会社(東京・品川)
- **製品**: WHILL Model C - 電動車いす、Model S - 歩道走行可能電動スクーター
- **Model S仕様**: 価格21万8,000円(非課税)、歩道走行可能、最高速度6km/h、航続距離約16km
- **特徴**: 自動車免許返納シニア向け、車のインターフェース踏襲(D/Rレバー)、楕円形ハンドル
- **ターゲット**: 65歳以上の約1,000万人が歩きづらさを感じる、年間60万人の免許返納者
- **バッテリー**: 鉛蓄電池(コスト抑制)、標準充電9時間40分
- **評価**: シニアカーの見た目問題を解決、おしゃれなデザインで心理的抵抗低減
- **URL**: https://whill.inc/

#### セグウェイジャパン(正規代理店)
- **製品**: Segway-Ninebotブランドの自立型電動スクーター、キックスクーター
- **用途**: テーマパーク、公園施設、大学内移動、警備、観光案内
- **シェアサービス**: IoT機器搭載、事業規模に合わせた小規模運用可能(10台~)
- **URL**: https://segway-japan.net/

### グローバルスタンダード

#### Segway Inc.(米国・中国Ninebot傘下)
- **歴史**: 2001年創業、2輪自立型パーソナルトランスポーター発明、2015年Ninebotに買収
- **製品**: Segway PT(Personal Transporter)、Ninebot miniシリーズ、電動キックスクーター
- **技術**: 自動バランス技術、大型空気入りタイヤ、強力モーター、最大65km航続距離
- **用途**: 空港、警備、観光地ツアー、倉庫・工場内移動
- **市場**: 世界シェアスクーターサービスで広く採用、市場シェア5%以上
- **URL**: https://www.segway.com/

#### Ninebot(中国)
- **概要**: Segway買収後、電動パーソナルモビリティ総合メーカーに
- **製品**: 自立バランススクーター、電動キックボード、ミニセグウェイ
- **特徴**: Bluetoothスピーカー内蔵、LEDライト、スマホアプリ連携
- **価格**: 消費者向け2-10万円、業務用10-50万円
- **URL**: https://www.ninebot.com/

#### 各国規制状況
- **米国**: 州・都市により異なる、一部で公道走行可能も規制強化傾向
- **欧州**: 歩道走行原則禁止、自転車レーン利用、速度制限6-20km/h
- **日本**: 原則私有地のみ、国土交通省認定エリア・実証実験地域で限定走行可

## 技術成熟度とハイプサイクル評価

**ハイプサイクル位置**: 安定期(ニッチ市場)
- 2001-2010年: 黎明期→期待のピーク(Segway発明、未来の移動手段として期待)
- 2010-2015年: 幻滅期(公道規制、価格高、普及せず)
- 2015-2020年: 啓蒙活動期(Ninebot買収、低価格化、業務用途確立)
- 2020年~: 安定期(観光・警備・介護等ニッチ市場で定着、大衆化せず)

**市場の再定義**:
- 当初の「革命的個人移動手段」から「特定用途向けツール」へ
- 電動車いす・シニアモビリティとして福祉分野で成長
- 観光地・施設内移動のレンタルサービスとして存続

## 日本の競争力分析 - 4つの要点

### 強み

1. **福祉・バリアフリー分野での応用**
   - WHILLが電動車いす・シニア向けスクーターで国際評価獲得
   - 日本の高齢化社会(65歳以上29%)を先取り、免許返納者の移動手段として需要
   - 介護保険適用・補助金制度により購入ハードル低減
   - バリアフリー設計、段差乗り越え、小回り性能等で日本メーカーの細やかな配慮

2. **規制整備と安全基準策定**
   - 歩道走行可能な電動車いす・シニアカーのISO規格化推進
   - 国土交通省が超小型モビリティ認定制度創設、実証実験地域指定
   - 道路交通法・道路運送車両法の整合性確保、段階的規制緩和

### 弱み

3. **グローバル市場での存在感欠如**
   - Segway(米国・中国)、Ninebot(中国)が世界市場を寡占、日本メーカーは追随
   - 自立バランス技術の特許で後発、価格競争力で中国勢に劣る
   - WHILL以外に国際競争力ある日本メーカー不在、市場シェア1%未満

4. **公道走行規制と普及の限界**
   - 日本は原則私有地・認定エリアのみ、公道走行ハードル高い
   - 電動キックボード(特定小型原付)が規制緩和される一方、立ち乗りスクーターは取り残され
   - 観光地・施設内レンタルに用途限定、日常移動手段として普及困難
   - 歩行者との事故リスク、保険制度未整備、社会受容性低い
