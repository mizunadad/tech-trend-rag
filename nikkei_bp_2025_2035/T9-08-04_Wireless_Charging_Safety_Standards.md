# T9-08-04 ワイヤレス給電の安全基準・電磁波規制

## Summary

無線給電技術の社会実装において、人体への電磁波影響、医療機器干渉、金属異物の誤加熱といった安全性確保が最優先課題となっている。IEC 61980国際規格とWHO電磁波ガイドラインに準拠した安全基準策定が進行中であり、総務省による周波数割当と電波法認証体制の整備が並行して推進されている。技術開発と規制整備の両輪により、2025年以降の本格的な市場展開が期待される。

### 5つの要点

1. **電磁波人体影響の安全評価**: ワイヤレス給電システムから発生する電磁波が人体、特にペースメーカーなどの医療機器に与える影響を定量評価し、WHOの国際非電離放射線防護委員会(ICNIRP)ガイドラインに準拠した安全基準を確立する必要がある。SAR(比吸収率)測定と電磁界強度評価により、人体への影響を数値的に検証する

2. **金属異物誤加熱防止技術**: 充電パッド上に硬貨、缶などの金属異物が存在すると、渦電流により発熱・発火リスクが生じる。異物検知センサー(FOD: Foreign Object Detection)により金属片を検出し、給電を自動停止する安全機構が必須となる。IEC 61980規格では異物検知精度と応答時間の要求仕様が定義されている

3. **通信機器干渉対策と周波数管理**: ワイヤレス給電の動作周波数(85kHz帯)が他の無線通信(AM放送、船舶通信、航空無線)に干渉しないよう、総務省が周波数割当と電波法認証基準を策定している。EMC(電磁両立性)試験により、通信機器への妨害波レベルを規制値以下に抑制する技術開発が進行中

4. **国際標準規格の策定と認証体制**: IEC 61980(電気自動車用ワイヤレス給電システム)、SAE J2954(米国自動車技術会規格)など、国際標準規格の策定が進展している。日本では電波法、電気用品安全法、道路運送車両法の三重規制に対応した認証体制の構築が課題となっている

5. **技術と規制の両輪による社会実装**: 安全技術の開発と法規制整備を同時並行で推進する産官学連携の枠組みが重要である。自動車メーカー、充電インフラ事業者、規制当局、学術機関が協働し、実証実験を通じて安全基準を継続的に更新するアジャイル型の規制アプローチが採用されている

## 具体的プロダクト事例

### 日本企業の先進事例

**1. 総務省 周波数割当・電波法認証制度**
- **URL**: https://www.soumu.go.jp/
- **概要**: ワイヤレス給電用周波数(85kHz、6.78MHz帯)の割当と電波法技術基準の策定を推進。2024年に電気自動車用ワイヤレス給電装置の技術基準を制定し、型式指定制度による認証体制を確立
- **技術的特徴**: 電界強度・磁界強度の測定方法を規定し、人体防護指針値(ICNIRP準拠)への適合性評価手法を標準化。異物検知機能の必須化により安全性を担保
- **品質保証の観点**: 電波法適合性試験(EMC試験、人体防護評価)の標準化により、製品認証プロセスの効率化と品質の均質化を実現

**2. デンソー FOD(異物検知)システム**
- **URL**: https://www.denso.com/
- **概要**: 充電パッド上の金属異物を高精度に検知し、給電を自動停止する安全システムを開発。磁界センサーアレイにより、硬貨サイズの小型金属片も検出可能
- **技術的特徴**: リアルタイム温度監視と異物検知を組み合わせ、発火リスクをゼロに近づける多重安全機構を実装。検知精度95%以上、応答時間100ms以下を達成
- **品質保証の観点**: 車載環境での長期信頼性試験(振動、温度サイクル、塩水噴霧)により、10年以上の耐久性を検証

**3. TDK ワイヤレス給電用EMCフィルタ**
- **URL**: https://www.tdk.com/
- **概要**: ワイヤレス給電システムから発生するノイズを抑制し、AM放送・車載通信機器への干渉を防止するフィルタコンポーネントを開発
- **技術的特徴**: コモンモードチョークコイルと高周波コンデンサを組み合わせた複合フィルタにより、85kHz帯のノイズを40dB以上減衰。小型・軽量化により車両への搭載性を向上
- **品質保証の観点**: CISPR 25(車載機器のEMC規格)準拠の試験により、ノイズレベルを規制値以下に抑制していることを定量的に検証

### グローバルスタンダード

**1. IEC 61980国際規格**
- **URL**: https://www.iec.ch/
- **概要**: 電気自動車用ワイヤレス給電システムの安全性・性能・互換性に関する国際標準規格。Part 1(一般要求事項)、Part 2(通信プロトコル)、Part 3(特定要求事項)で構成
- **技術的特徴**: 充電効率、電磁界曝露限界、異物検知性能、通信セキュリティの要求仕様を定義。SAR測定方法とファントム(人体模擬モデル)による評価手法を標準化
- **普及状況**: 2020年にPart 1が発行され、欧州・北米・アジアの主要自動車メーカーがこの規格に準拠した製品開発を推進中

**2. SAE J2954(米国自動車技術会規格)**
- **URL**: https://www.sae.org/
- **概要**: 3.7kW、7.7kW、11kWの3つの出力クラスに対応したワイヤレス給電システムの技術仕様を規定。北米市場で事実上の標準として採用
- **技術的特徴**: 85kHz動作周波数、Z軸方向の許容位置ずれ±75mm、効率85%以上を要求。FCC(米国連邦通信委員会)規制への適合性を明記
- **普及状況**: BMW、Mercedes-Benz、General Motorsなど欧米自動車メーカーの実証車両に採用

**3. ICNIRP電磁波ガイドライン**
- **URL**: https://www.icnirp.org/
- **概要**: WHO傘下の国際非電離放射線防護委員会が策定する電磁波曝露限界値。100kHz以下の低周波電磁界に対する人体防護指針を提供
- **技術的特徴**: 職業曝露(作業者)と一般公衆曝露で異なる制限値を設定。磁束密度、電界強度、SAR(比吸収率)の3つの指標で評価
- **普及状況**: 欧州、日本、オーストラリアなど多くの国が自国の電磁波規制の基準として採用

**4. WiTricity リファレンス設計**
- **URL**: https://witricity.com/
- **概要**: MIT発のワイヤレス給電技術ライセンス企業が提供する、SAE J2954準拠のリファレンス設計。自動車メーカーへの技術供与を通じて事実上の業界標準を形成
- **技術的特徴**: 磁界共鳴方式により、位置ずれに強く高効率な電力伝送を実現。異物検知、生体検知(Living Object Detection)機能を標準実装
- **普及状況**: トヨタ、日産、ホンダなど日本メーカーを含む30社以上にライセンス供与

## 技術動向の5つの特徴

1. **国際標準規格の確立による市場統一化**: IEC 61980とSAE J2954の2つの国際規格が並立しながらも、相互互換性を意識した仕様策定が進行している。これにより、グローバルサプライチェーンの形成と製品コストの低減が期待される

2. **人体安全性評価技術の高度化**: ファントム(人体等価モデル)を用いたSAR測定技術、数値シミュレーション(FDTD法)による電磁界解析技術が進化し、設計段階での安全性予測精度が向上している。実測とシミュレーションの組み合わせにより、開発期間の短縮とコスト削減を実現

3. **異物検知技術の多層化**: 磁界センサー、温度センサー、画像認識AIを組み合わせた多重安全機構により、検知精度と信頼性が飛躍的に向上。特にディープラーニングによる異物識別は、金属種類(鉄、アルミ、銅)の判別まで可能となり、誤検知率の低減に貢献

4. **EMC試験の自動化と効率化**: 電波暗室での手動測定から、マルチプローブシステムによる自動測定へ移行が進行。測定時間の短縮(従来48時間→8時間)により、開発サイクルの高速化と試験コストの削減を実現

5. **規制のアジャイル化と実証実験の推進**: 総務省の「新技術等に関する特例制度(規制のサンドボックス)」を活用し、規制を一時的に緩和した環境で実証実験を実施。その結果を規格改定にフィードバックする循環型の規制アプローチが定着しつつある

## 日本の競争力分析

### 強み

1. **厳格な電波法体系と認証ノウハウ**: 日本は電波利用が密集した環境であり、周波数調整と干渉対策に関する豊富な経験を有する。総務省の電波法技術基準は国際的にも高く評価され、IEC規格策定においてもリーダーシップを発揮

2. **車載部品メーカーの安全技術開発力**: デンソー、TDK、村田製作所など、車載電子部品で世界トップシェアを持つ日本企業が、異物検知センサー、EMCフィルタなどの安全部品開発で先行。長年の車載品質ノウハウ(AEC-Q規格対応)が競争優位の源泉

3. **医療機器メーカーとの連携によるペースメーカー干渉評価**: 日本光電、オムロンヘルスケアなどの医療機器メーカーと協働し、ペースメーカー装着者への影響評価を実施。臨床データに基づく安全基準策定が可能

4. **実証実験フィールドの充実**: 豊田市、横浜市など自治体と連携した実証実験エリアが整備され、実環境でのデータ収集と規格検証が進行中。「規制のサンドボックス」制度により、柔軟な実証実験環境を提供

### 弱み

1. **規制の複雑性と認証コスト**: 電波法、電気用品安全法、道路運送車両法の3つの法規制に対応する必要があり、認証取得に時間とコストがかかる。特に海外メーカーにとっては参入障壁となっており、国内市場の閉鎖性が懸念される

2. **国際標準化活動への参画不足**: IEC、IEEE、SAEなどの国際標準化機関への日本企業の参画は増加傾向にあるものの、欧米企業と比較すると議長・幹事ポストの獲得数が少ない。規格策定段階での発言力強化が課題

3. **ソフトウェア・AI技術者の不足**: 異物検知AIアルゴリズム、電磁界シミュレーションソフトウェア開発において、欧米のスタートアップ(WiTricity、Qualcomm等)に後れを取っている。ハードウェア偏重からソフトウェア・AI人材育成へのシフトが必要

4. **スタートアップエコシステムの脆弱性**: 安全規格・認証領域では、規制対応ノウハウと資金力を持つ大企業が優位であり、スタートアップの参入が困難。新興企業による破壊的イノベーションが起こりにくい構造
