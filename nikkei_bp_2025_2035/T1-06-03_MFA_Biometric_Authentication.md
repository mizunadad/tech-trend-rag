# T1-06-03 多要素認証（MFA）・生体認証統合

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 実用期（2024-2027年）
**技術成熟度**: TRL 8-9（商用展開・普及段階）

## Summary（5つの要点）

1. **「知識」「所持」「生体」の3要素**: パスワード（知識）だけに依存せず、スマートフォン・トークン（所持）、指紋・顔・虹彩（生体）を組み合わせた多層防御。1要素が破られても他の要素で防御し、なりすましを防止。

2. **FIDO2/WebAuthn標準の普及**: パスワードレス認証の国際標準。公開鍵暗号方式により、サーバー側にパスワードを保存しないため、漏洩リスクがゼロ。Apple Face ID、Windows Hello、Android指紋認証等が対応。

3. **リスクベース・適応型認証**: ログイン時の状況（場所、デバイス、時間帯、振る舞い）をAIで分析し、リスクに応じて認証強度を動的に変更。通常環境では簡易認証、異常時には追加認証を要求。

4. **生体認証精度の向上**: 指紋認証のFAR（False Acceptance Rate：他人受入率）は0.001%以下、顔認証は0.0001%以下まで向上。一卵性双生児でも区別可能。活性検知（生体検知）により、写真・模造品での突破を防止。

5. **プライバシー保護技術**: 生体情報をデバイス内で暗号化保存し、ネットワーク送信しない（オンデバイス認証）。Apple Secure Enclave、Google Titan M等の専用セキュリティチップにより、OS管理者でも生体情報にアクセス不可。GDPR、個人情報保護法に準拠。

## 具体的プロダクト事例

### 日本企業の先進事例

#### 1. **NTTドコモ「生体認証基盤（docomo Authenticator）」**
- **URL**: https://www.docomo.ne.jp/service/docomo_authenticator/
- **概要**: 顔・指紋・虹彩認証を統合したスマホ向け認証基盤。
- **技術特徴**: 
  - FIDO2準拠のパスワードレス認証
  - dアカウント（8,000万ユーザー）への適用
  - 金融機関、官公庁、企業向けAPI提供
- **導入実績**: みずほ銀行、三井住友銀行等の金融機関で採用

#### 2. **NEC「Bio-IDiom（バイオイディオム）」**
- **URL**: https://jpn.nec.com/biometrics/
- **概要**: NECの世界トップクラス生体認証技術群（顔・虹彩・指紋・静脈・声紋）。
- **技術特徴**: 
  - 顔認証精度：NIST評価で世界1位獲得（2018年）
  - 虹彩認証：誤認率0.00001%
  - マスク着用時も認証可能（2020年対応）
- **導入実績**: 
  - 全世界70カ国以上、1,000システム以上
  - 成田空港、羽田空港での顔認証ゲート
  - 新宿歌舞伎町での防犯カメラ統合システム

#### 3. **富士通「PalmSecure（手のひら静脈認証）」**
- **URL**: https://www.fujitsu.com/jp/group/frontech/solutions/business-technology/security/palmsecure/
- **概要**: 手のひらの静脈パターンで認証する非接触型生体認証。
- **技術特徴**: 
  - 他人受入率0.00008%、本人拒否率0.01%
  - 非接触（衛生的）、静脈は体内情報のため偽造困難
  - 経年変化に強い（指紋より安定）
- **導入実績**: 
  - 銀行ATM（全国8万台以上）
  - 企業入退室管理、PC/サーバーログイン
  - 医療機関（患者認証）

#### 4. **ソニー「UniPhier™ セキュアLSI」**
- **URL**: https://www.sony-semicon.com/ja/products/security.html
- **概要**: 生体認証データを保護する組み込みセキュリティチップ。
- **技術特徴**: 
  - ハードウェアベースの暗号化
  - タンパリング（改ざん）検知機能
  - FIDO2 Authenticator認証取得
- **適用分野**: スマートフォン、IoTデバイス、決済端末

### グローバルスタンダード

#### 1. **Apple「Face ID / Touch ID」**
- **URL**: https://support.apple.com/en-us/HT208108
- **概要**: iPhoneの3D顔認証（Face ID）と指紋認証（Touch ID）。
- **技術特徴**:

  - Secure Enclave（生体情報を暗号化保存、Apple社員でもアクセス不可）

- **URL**: https://www.microsoft.com/en-us/windows/windows-hello

- **技術特徴**: 
  - FIDO2準拠、パスワードレスログイン
  - TPM（Trusted Platform Module）2.0と統合
  - Azure Active Directory連携
- **普及**: Windows 10/11デバイス15億台以上（2024年）

#### 3. **Google「Pixel Face Unlock / Fingerprint」**
- **URL**: https://support.google.com/pixelphone/answer/6285273
- **概要**: Google Pixelスマートフォンの生体認証。
- **技術特徴**: 
  - Titan M2セキュリティチップ（生体情報保護）
  - Android FIDO2対応
  - Google Account（30億ユーザー）との統合
- **特徴**: オープンソースTrustZone技術を採用

#### 4. **Okta「Adaptive Multi-Factor Authentication」**
- **URL**: https://www.okta.com/products/adaptive-multi-factor-authentication/
- **概要**: リスクベース適応型MFAのグローバル標準。
- **技術特徴**: 
  - 7種類の認証要素（SMS、TOTP、Push通知、WebAuthn、生体認証等）
  - AIによるリスクスコアリング（場所、デバイス、行動パターン）
  - 18,000社以上の導入実績
- **統合**: 7,000以上のSaaS/アプリケーションと事前統合済み

## 全体要約（5つの要点）

### 1. 技術の本質
MFA・生体認証は「パスワードの終焉」を意味する。人間が記憶・管理する秘密情報（パスワード）から、変更不可能な生体情報と物理デバイス所持による認証へのシフト。フィッシング、辞書攻撃、ソーシャルエンジニアリング等の従来型攻撃を根本的に無効化。

### 2. コア技術要素
- **生体認証モダリティ**: 指紋（光学式・静電容量式・超音波式）、顔（2D/3D）、虹彩、静脈、声紋、歩容
- **暗号化技術**: 公開鍵暗号（RSA、ECC）、ハッシュ関数（SHA-256）、AES暗号
- **セキュリティチップ**: Apple Secure Enclave、Google Titan M、TPM 2.0、ARM TrustZone
- **活性検知**: 赤外線検知、3D深度センサー、ランダムチャレンジ応答
- **FIDO2/WebAuthn**: W3C国際標準、YubiKey等のハードウェアトークン

### 3. 応用分野
- **スマートフォン**: ロック解除、決済認証（Apple Pay、Google Pay）
- **金融**: ATM、オンラインバンキング、証券取引
- **企業**: PC/サーバーログイン、入退室管理、社内システムアクセス
- **官公庁**: マイナンバーカード、空港出入国審査、運転免許証
- **医療**: 患者認証、電子カルテアクセス、薬剤管理

### 4. 主要プレイヤー
- **米国**: Apple、Microsoft、Google、Okta、Yubico、Duo Security（Cisco）
- **日本**: NEC、富士通、NTTドコモ、日立、ソニー
- **欧州**: Gemalto（Thales）、IDEMIA、Fingerprint Cards（スウェーデン）
- **韓国**: Samsung（Knox）、LG

### 5. 将来展望
2026-2028年、FIDO2準拠デバイスが30億台を突破し、パスワードレス認証が主流に。脳波、心拍、DNA等の新モダリティ研究が進展。Generative AI（ディープフェイク）対策として、活性検知技術が高度化。2030年以降、連続認証（Continuous Authentication）により、ログイン後も継続的に本人確認し、セッションハイジャックを防止。

## 日本の立ち位置（4つの要点）

### 1. 強み：生体認証技術の世界トップレベル

NEC、富士通は生体認証技術で世界トップクラス。NECの顔認証はNIST（米国国立標準技術研究所）評価で世界1位獲得。富士通の手のひら静脈認証は全世界8万台以上のATMで稼働。日立の指静脈認証も高精度。日本企業は「精度」「安定性」「偽造防止」で強み。成田・羽田空港の顔認証ゲートは世界最大規模の実証事例。

### 2. 弱み：スマホ市場での出遅れ

生体認証技術で優位性があるにも関わらず、スマートフォン市場ではApple、Google、Samsung等の海外ベンダーが先行。日本の携帯キャリア（NTTドコモ、KDDI、ソフトバンク）は自社端末から撤退し、海外製スマホを販売。NEC、富士通の技術がスマホに搭載されず、B2B市場（企業・官公庁）に限定。スマホ14億台市場での存在感が薄い。

### 3. 機会：マイナンバーカード・デジタル庁政策

政府はマイナンバーカードの普及を推進（2024年で普及率76%）。将来的にスマホ搭載型マイナンバーカード、公的個人認証（JPKI）のFIDO2対応が検討され、日本企業の技術採用機会。デジタル庁は行政手続きのオンライン化を推進し、MFA必須化。金融庁もFISC安全対策基準でMFA推奨。国内市場拡大がベンダー育成の好機。

### 4. 脅威：ディープフェイク・合成メディア攻撃

Generative AI（Stable Diffusion、Midjourney等）の進化により、高精度な顔画像・音声合成が可能に。顔認証・声紋認証を突破するディープフェイク攻撃のリスクが増大。活性検知（赤外線、3D深度センサー、ランダムチャレンジ）の高度化が急務。プライバシー保護と偽造防止のバランスが課題。中国の顔認証監視社会への批判が高まり、欧米でプライバシー規制強化の動き。
