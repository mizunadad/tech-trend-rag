# T11-04-05 AI会話ロボット・回想法支援システム

## Summary（5つの要点）

1. **技術概要**: 認知症高齢者との対話を通じて記憶の想起を支援する回想法をAIロボットが実施。写真、音楽、映像を活用し、過去の記憶を刺激することで認知機能維持と精神的安定を促進
2. **実装技術**: 自然言語処理（NLP）、音声認識、感情認識AI、コンテンツ推薦アルゴリズムを統合。個人の人生史データベースと連携し、パーソナライズされた回想セッションを自動生成
3. **回想法の科学的根拠**: 認知症ケアの非薬物療法として国際的に認められた手法。長期記憶の活性化により短期記憶の低下を補完し、自己肯定感・QOL向上に寄与
4. **介護負担軽減**: 24時間対応可能なAIロボットが介護者の精神的・物理的負担を軽減。孤独感解消、徘徊抑制、BPSD（行動・心理症状）の改善に貢献
5. **データ蓄積と分析**: 会話内容、表情、音声トーンから認知機能・感情状態を継続的にモニタリング。AIが認知機能低下の兆候を早期検知し、家族・医療者に通知

## 品質保証エンジニアの視点

### 音声認識・感情認識の精度評価

**Python実装例: 音声認識精度評価**
```python
import pandas as pd
import numpy as np
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

def evaluate_speech_recognition(ground_truth_csv, recognition_results_csv):
    """
    AI会話ロボットの音声認識精度を評価
    
    Parameters:
    ground_truth_csv: 正解テキストのCSVファイル (列: utterance_id, true_text)
    recognition_results_csv: 音声認識結果のCSVファイル (列: utterance_id, recognized_text)
    
    Returns:
    評価指標: WER (Word Error Rate), 文字誤り率, 認識成功率
    """
    
    gt_df = pd.read_csv(ground_truth_csv)
    rec_df = pd.read_csv(recognition_results_csv)
    
    # データ結合
    df = pd.merge(gt_df, rec_df, on='utterance_id')
    
    def calculate_wer(reference, hypothesis):
        """Word Error Rate計算"""
        ref_words = reference.split()
        hyp_words = hypothesis.split()
        
        # Levenshtein距離計算（簡易版）
        d = np.zeros((len(ref_words) + 1, len(hyp_words) + 1))
        
        for i in range(len(ref_words) + 1):
            d[i][0] = i
        for j in range(len(hyp_words) + 1):
            d[0][j] = j
            
        for i in range(1, len(ref_words) + 1):
            for j in range(1, len(hyp_words) + 1):
                if ref_words[i-1] == hyp_words[j-1]:
                    d[i][j] = d[i-1][j-1]
                else:
                    substitution = d[i-1][j-1] + 1
                    insertion = d[i][j-1] + 1
                    deletion = d[i-1][j] + 1
                    d[i][j] = min(substitution, insertion, deletion)
        
        return d[len(ref_words)][len(hyp_words)] / len(ref_words)
    
    # WER計算
    df['wer'] = df.apply(lambda row: calculate_wer(row['true_text'], row['recognized_text']), axis=1)
    
    # 文字誤り率計算
    def char_error_rate(reference, hypothesis):
        ref_chars = list(reference.replace(' ', ''))
        hyp_chars = list(hypothesis.replace(' ', ''))
        
        d = np.zeros((len(ref_chars) + 1, len(hyp_chars) + 1))
        
        for i in range(len(ref_chars) + 1):
            d[i][0] = i
        for j in range(len(hyp_chars) + 1):
            d[0][j] = j
            
        for i in range(1, len(ref_chars) + 1):
            for j in range(1, len(hyp_chars) + 1):
                if ref_chars[i-1] == hyp_chars[j-1]:
                    d[i][j] = d[i-1][j-1]
                else:
                    d[i][j] = min(d[i-1][j-1] + 1, d[i][j-1] + 1, d[i-1][j] + 1)
        
        return d[len(ref_chars)][len(hyp_chars)] / len(ref_chars)
    
    df['cer'] = df.apply(lambda row: char_error_rate(row['true_text'], row['recognized_text']), axis=1)
    
    # 完全一致率
    df['exact_match'] = df['true_text'] == df['recognized_text']
    exact_match_rate = df['exact_match'].mean()
    
    return {
        'mean_wer': df['wer'].mean(),
        'median_wer': df['wer'].median(),
        'mean_cer': df['cer'].mean(),
        'exact_match_rate': exact_match_rate,
        'samples_evaluated': len(df)
    }

# 使用例

# print(f"平均WER: {results['mean_wer']:.3f}")
# print(f"完全一致率: {results['exact_match_rate']:.1%}")
```

**感情認識精度評価**
```python
from sklearn.metrics import confusion_matrix, classification_report
import seaborn as sns
import matplotlib.pyplot as plt

def evaluate_emotion_recognition(ground_truth_csv, emotion_results_csv):
    """
    AI会話ロボットの感情認識精度を評価
    
    Parameters:
    ground_truth_csv: 正解感情ラベルのCSVファイル (列: utterance_id, true_emotion)
    emotion_results_csv: 感情認識結果のCSVファイル (列: utterance_id, predicted_emotion, confidence)
    
    Returns:
    混同行列、精度・再現率・F1スコア
    """
    
    gt_df = pd.read_csv(ground_truth_csv)
    pred_df = pd.read_csv(emotion_results_csv)
    
    df = pd.merge(gt_df, pred_df, on='utterance_id')
    
    # 感情カテゴリ
    emotion_labels = ['happy', 'sad', 'angry', 'neutral', 'anxious']
    
    # 混同行列
    cm = confusion_matrix(df['true_emotion'], df['predicted_emotion'], labels=emotion_labels)
    
    # 精度評価
    accuracy = accuracy_score(df['true_emotion'], df['predicted_emotion'])
    
    # 詳細レポート
    report = classification_report(df['true_emotion'], df['predicted_emotion'], 
                                    labels=emotion_labels, output_dict=True)
    
    # 可視化
    plt.figure(figsize=(10, 8))
    sns.heatmap(cm, annot=True, fmt='d', cmap='Blues', 
                xticklabels=emotion_labels, yticklabels=emotion_labels)
    plt.title('Emotion Recognition Confusion Matrix')
    plt.ylabel('True Emotion')
    plt.xlabel('Predicted Emotion')
    plt.savefig('emotion_confusion_matrix.png')
    
    return {
        'accuracy': accuracy,
        'confusion_matrix': cm,
        'classification_report': report,
        'per_emotion_f1': {label: report[label]['f1-score'] for label in emotion_labels}
    }

# 使用例
# results = evaluate_emotion_recognition('emotion_ground_truth.csv', 'emotion_predictions.csv')

# print(f"各感情のF1スコア: {results['per_emotion_f1']}")
```

### 製品品質評価への応用

1. **音声認識の頑健性評価**: 高齢者特有の発話（語尾不明瞭、方言、聴覚障害による大声）での認識精度検証
2. **感情認識の精度**: 表情・音声トーンから感情を正しく推定できるか、誤認識による不適切な応答の防止
3. **会話の自然性**: 回想法セッションの会話フローが自然か、唐突な話題転換がないか、ユーザー満足度評価
4. **長期使用での性能劣化**: 長期間使用後の音声認識精度低下、学習データの偏り、定期キャリブレーションの必要性評価

### メカトロニクス製品への応用

1. **製造ラインの作業者状態監視**: 音声・表情から疲労・ストレス状態を検出、安全管理・生産性向上に活用
2. **製品検査員の集中度モニタリング**: 検査作業中の注意力低下を検知、ミス防止のためのアラート発信
3. **顧客サポートAIの品質評価**: 問い合わせ対応AIの音声認識・感情認識精度を定量評価、顧客満足度向上
