## T18-02-02 ブルー水素製造技術（CCS併用）

### 💡 全体要約の特徴
天然ガスや石炭などの化石燃料から水素を製造する際に発生する**CO2を回収・貯留（CCS: Carbon Capture and Storage）**することで、実質的なCO2排出量を大幅に削減する技術。グリーン水素よりも**低コスト**で製造可能であり、既存の天然ガスインフラを活用できるため、脱炭素移行期における**現実的な選択肢**とされる。日本ではINPEX、JERAなどがCCSの実証実験を進めている。ただし、CO2回収率が100%ではないため、**完全なゼロエミッションではない**点が課題として残る。

#### 概念図: ブルー水素製造とCCSのプロセス
```mermaid
graph TD
    A[天然ガス/石炭 (化石燃料)] --> B{改質・ガス化 (水素製造)}
    B --> C[水素 (H2)]
    B --> D[CO2 発生]
    D --> E{CO2 分離・回収 (CCS)}
    E --> F[地下貯留 (長期安定貯留)]
    C --> G[貯蔵・輸送/利用]
```

### ⚖️ 日本の立ち位置分析（強み・弱み）
| 項目 | 強み (Strength) | 弱み (Weakness) |
| :--- | :--- | :--- |
| **技術** | CO2分離回収技術（特に膜分離）と地下貯留技術の研究実績。 | CO2回収・輸送・貯留インフラ整備に**巨額の初期投資**が必要。 |
| **市場** | 既存の天然ガスインフラを活かせるため、**早期の大量供給**が可能。 | 化石燃料への依存が継続することに対する**環境保護団体からの懸念**。 |

### 📈 ハイプ・サイクル評価と5つの要点Summary
| 評価項目 | 予測/状況 |
| :--- | :--- |
| **現状** | 安定期へ近づく（移行期の重要技術として確立） | 
| **到達度** | 高い | 
| **普及率** | 中 | 

**5つの要点Summary:**
1. **技術定義**: 天然ガス等から水素を製造し、発生CO2を**CCSで回収・貯留**する。
2. **コスト優位性**: グリーン水素に比べ**低コスト**であり、脱炭素移行期に大量供給を担う。
3. **既存インフラ活用**: 既存の天然ガス関連インフラを流用できるため、**早期の社会実装**が容易。
4. **日本の動向**: INPEX、JERAなど日本企業が**CCS実証**を推進中。
5. **課題**: **完全なゼロエミッションではない**点と、CO2貯留場所の選定・安全性確保。
