# T7-03-05 自動運転・無人搬送システム（AGV/AMR）

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 生産性の安定期（2025年時点）
**市場成熟度**: 実用化完了～高度化・普及拡大段階

### Summary（5つの要点）

1. **AGVからAMRへの進化**: 従来のAGV(床面ガイド追従型)から、AI・センサー自律走行のAMR(自律移動ロボット)へ進化。障害物回避、動的経路再計画が可能に
2. **物流倉庫での大規模導入**: Amazon、楽天、ZOZO等がロボット倉庫を構築。人間の歩行時間削減、24時間稼働により物流生産性を劇的向上
3. **製造ラインでの標準装備化**: トヨタ、日産等の自動車製造、パナソニック、ソニー等の電機製造で部品搬送を無人化。ジャストインタイム生産を支える基盤技術
4. **日本企業の世界トップシェア**: 村田機械、ダイフク、IHI等の日本企業がAGV/AMR製造で世界シェアトップクラス。FA(ファクトリーオートメーション)技術の蓄積が強み
5. **協働ロボットとの統合**: 搬送ロボット(AGV/AMR) + 作業ロボット(協働ロボット)の統合により、「取りに行く→作業する→運ぶ」の一連の流れを完全無人化

## 全体要約（5つの要点）

1. **技術の本質**: 工場・倉庫内の部品・製品搬送を無人化するロボットシステム。AGV(ガイド追従型)からAMR(自律走行型)へ進化し、AI・センサーで障害物回避・経路最適化が可能に
2. **主要応用分野**: 物流倉庫(EC、3PL)、製造ライン(自動車、電機、食品)、病院(薬品・検体搬送)、ホテル(リネン搬送)、空港(手荷物搬送)など多様な分野で実用化
3. **技術的進化方向**: ①AGV→AMRへ自律化、②単独動作→複数台協調制御(フリート管理)、③搬送のみ→作業ロボット統合、④屋内専用→屋外配送へ拡大
4. **市場動向**: 世界市場は2030年に約200億米ドル予測。Amazon等の大規模導入が市場牽引。日本では村田機械、ダイフク等が世界トップシェア維持
5. **将来展望**: 搬送ロボット + 協働ロボット + AIによる完全無人工場・倉庫の実現。屋外配送ロボット(ラストワンマイル)の実用化により、物流の完全無人化へ

## 日本の立ち位置：強み・弱み分析（4つの要点）

### 1. AGV/AMR製造での世界トップシェアと技術力

**強み**
- 村田機械(半導体工場向けAGV世界シェア1位)、ダイフク(物流システム世界シェア1位)等が世界市場を牽引
- 日本の製造業が培った高精度位置決め、クリーンルーム対応技術で差別化
- トヨタ生産方式(ジャストインタイム)とAGVの親和性が高く、製造ラインでの実証蓄積
- 豊田自動織機のフォークリフトAGVが世界標準として普及

**弱み**
- 物流倉庫向け統合ソリューション(AGV + 自動倉庫 + WMS)ではAmazon Roboticsに大きく後れ
- 低価格帯AMRで中国Geek+等が急成長し、価格競争で劣勢
- AI自律走行技術(SLAM、障害物認識)では米国・欧州企業に技術的に遅れ
- オープン標準規格(VDA5050等)への対応が遅れ、メーカー間相互運用性が低い

### 2. 製造業での実用化先行と物流への展開遅れ

**強み**
- 自動車製造(トヨタ、日産、ホンダ)、電機製造(パナソニック、ソニー)でAGV導入が早期に進展
- 日本の製造業の品質要求(位置精度±1mm、クリーンルーム対応)に対応可能な高品質AGV
- 製造DX、スマートファクトリー推進の一環としてAGV導入が加速
- 協働ロボット(ファナック、安川電機、川崎重工業)とAGVの統合で新たな競争力創出

**弱み**
- EC物流倉庫(Amazon、楽天)では海外製AMRの導入が先行し、国産AGV/AMRの採用が遅れ
- 物流業界の慎重姿勢(「人間の方が柔軟」という固定観念)により、AGV導入が製造業ほど進まず
- WMS(倉庫管理システム)、TMS(輸送管理システム)とAGVの統合が複雑で、導入ハードルが高い
- 中小物流企業でのAGV導入率が低く(5%未満)、大手物流との格差が拡大

### 3. AI自律走行技術での海外依存と屋外配送遅れ

**強み**
- ZMP、Rapyuta Robotics等のロボットベンチャーがAMR開発で先行、実証実験を積極展開
- 経産省、国交省が公道配送ロボット実証プロジェクトを支援、規制緩和を推進
- 日本の高精度3D地図(ダイナミックマップ基盤)がAMR屋外走行に活用可能
- セブン-イレブン、ローソン等のコンビニ配送ロボット実証で先行事例蓄積

**弱み**
- 屋外配送ロボット(ラストワンマイル)では米国Starship Technologies、中国Meituan等に大きく後れ
- 公道走行規制が厳しく(道路交通法、歩道走行制限)、実用化が遅れる
- AI自律走行の基盤技術(LiDAR、カメラ画像認識、SLAM)では海外製センサー・アルゴリズムに依存
- ドローン配送では中国・米国に先行され、陸上配送ロボットでも同様の懸念

### 4. 協働ロボット統合と完全無人工場への展開

**強み**
- ファナック、安川電機、川崎重工業等の産業用ロボットメーカーが協働ロボット開発で世界トップクラス
- AGV(搬送) + 協働ロボット(組立・検査)の統合により、「運ぶ→作業する」の一連の流れを無人化
- トヨタの未来工場(Woven City)で完全無人生産ラインの実証進行中
- 日本のロボティクス技術(精密制御、安全制御)が統合システムで競争力を発揮

**弱み**
- 搬送ロボット + 作業ロボットの統合制御プラットフォームが未成熟で、個別開発に留まる
- メーカー間の相互運用性が低く、異なるメーカーのAGVとロボットの統合が困難
- AI統合制御(複数ロボットの協調動作)では米国Boston Dynamics、ドイツKUKA等に技術的に遅れ
- 完全無人工場の実現には、品質検査、メンテナンス、異常対応も無人化が必要で、まだ課題多数

## 品質保証エンジニアとしての活用視点

### メカトロニクス製品品質業務への応用

1. **工場内物流の品質向上**
   - AGV/AMRによる部品搬送の無人化により、搬送時の損傷・取り違えを削減
   - 搬送履歴の自動記録により、トレーサビリティを強化
   - 人間搬送時の「うっかりミス」(部品置き忘れ、取り違え)を完全排除

2. **製造リードタイムの短縮と在庫適正化**
   - AGV/AMRによる24時間稼働により、製造リードタイムを短縮
   - ジャストインタイム生産の精度向上、在庫削減、キャッシュフロー改善
   - 生産計画の柔軟性向上、需要変動への迅速対応

3. **品質データ自動収集・トレーサビリティ強化**
   - AGV/AMRが搬送する部品・製品にRFID/バーコードを自動読取
   - 「どの部品が」「いつ」「どこで」使われたかを自動記録
   - 不具合発生時の影響範囲特定(どの製品に使われたか)を迅速化

4. **クリーンルーム・無塵環境での品質保証**
   - 半導体・液晶製造のクリーンルームで人間の立ち入りを最小化
   - AGVによる部品搬送により、人間由来の塵埃発生を削減
   - クリーン度維持により、製品品質の安定化

5. **協働ロボット + AGVによる検査工程自動化**
   - AGVが検査対象製品を運搬、協働ロボットが外観検査・寸法測定を実施
   - 検査完了後、AGVが次工程へ自動搬送
   - 検査工程の完全無人化により、24時間検査体制を実現
