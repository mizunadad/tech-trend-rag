# T1-01-05 統合セキュリティ・監視システム

## ハイプ・サイクル位置づけ

**現在のフェーズ**: 実用期初期（2026-2028年）
**技術成熟度**: TRL 7-8（実証実験から商用展開段階）

## Summary（5つの要点）

1. **物理セキュリティ（防犯カメラ・センサー）とサイバーセキュリティの統合管理**: 従来は別々に管理されていた「物理セキュリティ（侵入検知、監視カメラ、入退管理）」と「サイバーセキュリティ（ネットワーク監視、マルウェア対策）」を一元管理。都市全体のセキュリティ状況を統合的に把握し、物理攻撃とサイバー攻撃の複合脅威に対応。

2. **AI画像解析による自動脅威検知**: 数千～数万台の防犯カメラ映像をAIがリアルタイム解析。不審者検知、置き去り荷物検知、群衆異常行動検知、車両ナンバープレート認識を自動化。人手監視では不可能な大規模監視を実現し、犯罪抑止と早期対応を両立。

3. **ゼロトラストアーキテクチャ**: 「内部ネットワークも信頼しない」という原則に基づき、すべてのアクセスを認証・認可・検証。VPN境界防御の限界を克服し、テレワーク時代のサイバーセキュリティを実現。マイクロセグメンテーション、多要素認証、継続的監視が柱。

4. **SOC/SOAR統合**: SOC（Security Operations Center：セキュリティ監視センター）とSOAR（Security Orchestration, Automation and Response：自動対応）を統合。AIがセキュリティイベントを自動分析し、優先度判定、初動対応、エスカレーションを自動実行。人手不足を補い、インシデント対応時間を大幅短縮。

5. **プライバシー保護技術との両立**: 防犯カメラ映像のエッジAI処理（顔をぼかし、行動パターンのみ抽出）、差分プライバシー、仮名化処理等により、市民のプライバシーを保護しながらセキュリティを確保。GDPR、個人情報保護法等の規制に準拠。

## 全体要約（5つの要点）

### 1. 技術の本質
統合セキュリティ・監視システムは「物理空間とサイバー空間の両面から都市・組織を守る統合防御基盤」である。防犯カメラ、センサー、入退管理（物理セキュリティ）と、ネットワーク監視、マルウェア対策、認証管理（サイバーセキュリティ）を一元管理し、複合的脅威に対応。AIによる自動脅威検知・対応により、人手不足を補完し、インシデント対応時間を短縮。

### 2. コア技術要素
- **AI画像解析**: 物体検知、顔認証、行動解析、群衆異常検知
- **ゼロトラストアーキテクチャ**: マイクロセグメンテーション、多要素認証、継続的監視
- **SIEM/SOAR**: セキュリティイベント統合管理、自動対応
- **生体認証**: 指紋、顔、虹彩、静脈認証
- **プライバシー保護**: エッジAI、差分プライバシー、仮名化

### 3. 応用分野
- **スマートシティ**: 公共空間の安全確保、犯罪抑止
- **重要インフラ**: 発電所、空港、駅、政府施設のセキュリティ
- **企業**: オフィス、工場、物流倉庫の物理・サイバーセキュリティ統合
- **イベント**: オリンピック、万博、コンサート等の大規模イベントセキュリティ
- **金融**: 銀行、証券のサイバー攻撃対策、店舗防犯

### 4. 主要プレイヤー
- **日本**: セコム、ALSOK、NTTデータ、日立、パナソニック、NEC
- **米国**: Palantir、Palo Alto Networks、Cisco、Microsoft、Genetec（加）
- **欧州**: Axis Communications（瑞）、Darktrace（英）、Bosch（独）
- **中国**: Hikvision、Dahua、ZTE

### 5. 将来展望
2026-2028年、ゼロトラストアーキテクチャが企業・自治体で標準化され、VPN境界防御からの移行が加速。AI画像解析が高度化し、プライバシー保護技術（エッジAI、自動マスキング）との両立が進む。2028-2030年、SOARによる完全自動対応が実用化され、セキュリティ人材不足を技術で補完。Beyond 5G/6G通信基盤の整備により、数百万台のIoTセキュリティセンサーがリアルタイム統合される。2030年以降、量子暗号通信が実用化され、「破られない通信」が実現。AI対AI（攻撃側と防御側の両方がAI）の時代となり、自律型防御システムが主流となる。

## 日本の立ち位置（4つの要点）

### 1. 強み：物理セキュリティとハードウェア技術

日本企業（セコム、ALSOK）は、物理セキュリティ（警備サービス、防犯カメラ、センサー）で高いシェアを持つ。セコムは日本国内の警備業界トップであり、24時間監視体制、緊急時の警備員派遣等の総合サービスを提供。東京オリンピック2021では、ALSOKがセキュリティシステムを提供し、実績を積んだ。

パナソニック、日立、NECは、AI画像解析、生体認証（顔認証、指紋認証、静脈認証）等のハードウェア技術で強みを持つ。特に、パナソニックのAI防犯カメラ、日立の指紋・静脈認証システムは、世界的にも評価が高い。空港、駅、スタジアム等の公共施設で採用実績がある。

日本は、プライバシー保護への意識が高く、エッジAIによる映像解析（顔をぼかし、行動パターンのみ抽出）、自動マスキング等の技術開発が進んでいる。欧州GDPR、日本の個人情報保護法に準拠したプライバシー保護機能は、グローバル展開の際の強みとなる。

### 2. 弱み：サイバーセキュリティの海外依存と人材不足

日本のサイバーセキュリティ市場は、米国企業（Palo Alto Networks、Cisco、Microsoft、CrowdStrike等）に大きく依存している。ファイアウォール、侵入検知システム（IDS/IPS）、エンドポイント保護（EDR）等の主要製品は、ほぼ海外製である。国産のサイバーセキュリティ製品は、NTTデータ、NEC、トレンドマイクロ（日本起源だが現在は米国法人）等が提供しているが、世界シェアは限定的。

ゼロトラストアーキテクチャの導入が遅れている。米国では、2021年の大統領令により連邦政府機関へのゼロトラスト導入が義務化されたが、日本では2023年頃からようやく大手企業で導入が始まった段階。中小企業ではVPN境界防御が主流であり、テレワーク時代のセキュリティリスクを抱えている。

セキュリティ人材が絶対的に不足している。経済産業省の調査では、2030年に約19万人のセキュリティ人材が不足すると予測されている。SOC（セキュリティ監視センター）の運用人材、インシデント対応人材、ペネトレーションテスター等が特に不足しており、海外人材の活用、AI自動化による補完が急務。

統合セキュリティプラットフォーム（物理+サイバー）の実装が遅れている。セコム、ALSOKは物理セキュリティ中心であり、サイバーセキュリティとの統合は発展途上。NTTデータ、日立は統合ソリューションを提供しているが、海外のGenetec、Palantir等の高度なプラットフォームと比較すると、機能・実績で劣る。

### 3. 機会：2025年大阪万博とサイバーセキュリティ基本法

2025年の大阪・関西万博は、統合セキュリティシステムの大規模実証の場となる。会場全体の防犯カメラ、入退管理、サイバーセキュリティを統合管理し、AI自動脅威検知、プライバシー保護技術を実装。成功すれば、国際的な注目を集め、技術輸出の足がかりとなる。

日本政府は「サイバーセキュリティ基本法」を改正し、重要インフラ（電力、通信、金融、交通等）へのサイバーセキュリティ対策強化を義務化。これにより、企業・自治体のセキュリティ投資が拡大する。特に、ゼロトラストアーキテクチャ、SOC/SOAR導入が加速する見込み。

ASEAN諸国、インド等では、都市化・デジタル化が進む一方で、サイバー攻撃、犯罪が増加している。日本の統合セキュリティ技術（物理+サイバー）は、これらの国々への輸出ポテンシャルが高い。JICAのODAプロジェクトと連携した技術協力も検討されている。

量子暗号通信の研究開発で、日本（東芝、NTT、NICT）は世界トップクラスの技術を保有している。2030年頃の実用化を見据え、量子セキュリティ技術の国際標準化・市場獲得を目指す。

### 4. 脅威：中国の国家監視システムと米国のプラットフォーム支配

中国は、「天網（Skynet）」「雪亮工程」等の国家規模の監視システムを構築しており、Hikvision、Dahua等の企業が世界シェアを拡大している。中国製の防犯カメラは低価格で高性能であり、途上国を中心に急速に普及。日本企業が価格競争で不利な立場に置かれている。

ただし、中国製カメラは「バックドア（不正アクセス経路）」のリスクが指摘されており、米国政府は調達を禁止、日本政府も公共調達で制限する方向。プライバシー侵害・人権問題への懸念もあり、欧米諸国では中国製品の排除が進んでいる。日本企業は、「信頼性・プライバシー保護」を強みとして差別化する戦略が必要。

米国企業（Palo Alto Networks、Microsoft、Cisco、Palantir等）は、サイバーセキュリティ市場を支配しており、日本企業がこれらのプラットフォームに依存する構造が固定化している。特に、Microsoft Azureのエコシステムに組み込まれると、国産技術の差別化が困難になる。

AI対AIの時代において、攻撃側（マルウェア、フィッシング）もAIを活用し、高度化・自動化が進んでいる。日本のセキュリティ企業は、AI技術開発で米国・中国に後れを取っており、防御側のAI技術強化が急務。

また、統合セキュリティシステムは、膨大なデータ（映像、ネットワークログ、認証ログ等）を収集・分析するため、プライバシー侵害のリスクが常に存在する。欧州GDPR、日本の個人情報保護法、中国のデータセキュリティ法等、各国の規制が異なり、グローバル展開の際に複雑なコンプライアンス対応が必要となる。市民社会からの監視社会化への批判も強く、技術導入と社会受容性のバランスが課題。
